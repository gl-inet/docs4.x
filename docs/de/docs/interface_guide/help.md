# Hilfe

Klicken Sie im Web-Admin-Panel auf das Fragezeichen-Symbol auf der rechten Seite der Titelleiste.

![help](https://static.gl-inet.com/docs/router/en/4/interface_guide/help/help.png){class="glboxshadow"}

Dort finden Sie die [Router Support Docs](https://docs.gl-inet.com/router/en/4){target="_blank"}.

Seit Firmware v4.8 können Sie den Tech Support mit nur einem Klick kontaktieren, wodurch es einfacher ist, Unterstützung zu erhalten.

Klicken Sie auf **Contact Tech Support**. Sie werden zur Log-Seite weitergeleitet, und auf der rechten Seite erscheint ein Pop-up-Fenster.

![contact support](https://static.gl-inet.com/docs/router/en/4/interface_guide/help/contact_support.png){class="glboxshadow"}

![send feedback](https://static.gl-inet.com/docs/router/en/4/interface_guide/help/send_feedback.png){class="glboxshadow"}

Die Optionen Upload System Log und Upload Debug Info sind standardmäßig aktiviert.

Füllen Sie die erforderlichen Informationen aus und klicken Sie für Unterstützung auf **Send**.
