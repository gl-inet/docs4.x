# Wie verbindet man ein Gerät mit ausschließlich Ethernet-Anschluss über einen GL.iNet Router mit WLAN?

Wenn Ihr Gerät nur eine Ethernet-Verbindung unterstützt, aber eine Verbindung zu einem drahtlosen Netzwerk herstellen muss, kann ein GL.iNet Router diese Lücke schließen.

Ein GL.iNet Router kann für Geräte mit ausschließlich kabelgebundener Verbindung als Ethernet-zu-WLAN-Brücke dienen und ihnen so den Beitritt zu einem drahtlosen Netzwerk ermöglichen. Dazu wird der Router als Repeater konfiguriert.

Bitte verbinden Sie Ihr Gerät mit dem LAN-Port des Routers und richten Sie dann den [Repeater](../interface_guide/internet_repeater.md) ein.
