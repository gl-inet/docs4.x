# DPIライセンス

**注意**：この機能は現に **GL-MT5000 (Brume 3)** でのみ利用可能です。

DPI（Deep Packet Inspection）はインテリジェントネットワーク管理のコア機能です。従来のルーター（送信元または宛先アドレスのみを特定）の制限を克服し、データペイロードを深く分析し、特徴ライブラリの比較を通じてユーザーがアクセスしたアプリケーションとウェブサイトを正確に特定し、洗練されたトラフィック分類と制御を可能にします。

[Netify](https://www.netify.ai/){target="_blank"} と統合され、GL.iNet DPI機能は効率のな展開のための軽量組み込みプラグインを採用しています。Netifyのオンラインでアップデートされる署名データベースにより、信頼性の高い管理を可能にし、ネットワーク制御をより正確かつ効率のにします。

ルーターのWeb管理パネルにログインし，FLOW CONTROL > DPIライセンスに移動します。

**30日間試用版**を開始して、**データ統計**、**コンテンツフィルター**、**QoS**を含むDPI機能をお楽しみください。
