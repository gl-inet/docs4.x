# ¿Cómo conectar un router GL.iNet a una antena Starlink?

Si necesita conectar un router GL.iNet a una antena Starlink, siga los pasos que se indican a continuación.

**Topología**

![starlinkgl](https://static.gl-inet.com/docs/router/en/4/faq/starlink/starlinkgl.jpg){class="glboxshadow"}

1. Conecte la alimentación del router Starlink a la antena Starlink.

2. Conecte el puerto LAN del router Starlink al puerto WAN del router GL.iNet.

3. Conecte un dispositivo al puerto LAN o al Wi-Fi del router GL.iNet y, a continuación, configure el router para el acceso a Internet. [Configuración inicial](first_time_setup.md)

   **Nota**: Si su router GL.iNet admite [Network Acceleration](../interface_guide/network_acceleration.md), vaya al panel de administración web -> **NETWORK** -> **Network Acceleration** y desactívelo.

   ![netacc](https://static.gl-inet.com/docs/router/en/4/faq/starlink/netacc.jpg){class="glboxshadow"}

4. Abra la aplicación Starlink, vaya a Settings y habilite **Bypass mode**.
