# Erweiterte Einstellungen

Gehen Sie auf der linken Seite des Web-Admin-Panels zu **SYSTEM** -> **Advanced Settings**.

Sie können erweiterte Einstellungen mit LuCI ändern, der standardmäßigen Weboberfläche von OpenWrt.

![advanced settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/go_to_luci.png){class="glboxshadow"}

LuCI ist ein offenes und unabhängiges Projekt, das von OpenWrt gepflegt wird, und wird ohne zusätzliche Gewähr bereitgestellt. GL.iNet ist nicht für die Wartung von LuCI verantwortlich.

Klicken Sie auf die Schaltfläche **Go To LuCI**; Sie werden dann zur LuCI-Anmeldeseite weitergeleitet. 

![luci login](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/luci_login.jpg){class="glboxshadow"}

Geben Sie das Anmeldepasswort ein. Es ist identisch mit dem Passwort des Web-Admin-Panels.

Anschließend werden Sie bei LuCI angemeldet. Dies ist ein Beispiel der LuCI-Seite für den GL-AX1800 (Flint).

![luci logged in](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/luci_logged_in.jpg){class="glboxshadow"}
