# Pomoc

W panelu administracyjnym WWW kliknij ikonę znaku zapytania po prawej stronie paska tytułu.

![help](https://static.gl-inet.com/docs/router/en/4/interface_guide/help/help.png){class="glboxshadow"}

Znajdziesz tam [dokumentację wsparcia routerów](https://docs.gl-inet.com/router/en/4){target="_blank"}.

Od firmware v4.8 możesz skontaktować się z pomocą techniczną jednym kliknięciem, co ułatwia uzyskanie wsparcia.

Kliknij **Contact Tech Support** — zostaniesz przeniesiony na stronę **Log**, a po prawej stronie pojawi się wyskakujące okno.

![contact support](https://static.gl-inet.com/docs/router/en/4/interface_guide/help/contact_support.png){class="glboxshadow"}

![send feedback](https://static.gl-inet.com/docs/router/en/4/interface_guide/help/send_feedback.png){class="glboxshadow"}

Opcje **Upload System Log** i **Upload Debug Info** są domyślnie zaznaczone.

Wypełnij wymagane informacje i kliknij **Send**, aby uzyskać pomoc.
