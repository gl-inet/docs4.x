# ¿Cómo conectar a Wi-Fi un dispositivo que solo dispone de Ethernet con un router GL.iNet?

Si su dispositivo solo admite conectividad Ethernet, pero necesita conectarse a una red inalámbrica, un router GL.iNet puede cubrir esa necesidad.

Un router GL.iNet puede actuar como un puente de Ethernet a Wi-Fi para dispositivos que solo tienen conexión por cable, permitiéndoles unirse a una red inalámbrica. Esto se logra configurando el router como Repeater.

Conecte su dispositivo al puerto LAN del router y luego configure [repeater](../interface_guide/internet_repeater.md).
