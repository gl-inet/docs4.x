GL.iNet の製品およびサービス（以下「GL.iNet」または「当社」といいます）をご利用いただき、ありがとうございます。

お客様が直面している問題の解決をより適切に支援するため、当社の技術サポートチームは AnyDesk ソフトウェアを使用してリモート接続を行い、問題の原因を特定したうえで、的確なサポートを提供します。

## I. リモート接続の目的

当社は、GL.iNet の製品およびサービスに関する技術的な問題の解決を支援する目的に限って、リモート接続を使用します。これには、以下が含まれますが、これらに限定されません。

1. 問題の診断およびトラブルシューティング

2. GL.iNet の製品およびサービスの設定確認および変更

当社は、GL.iNet の製品およびサービスに関する技術サポートと無関係なお客様の個人情報、ファイル、その他のデータを閲覧、収集、保存、または配布しないことをお約束します。また、お客様の同意なくソフトウェアをインストールすることもありません。

## II. 許可される範囲

トラブルシューティングを円滑に行うため、お客様は当社に対し、以下を許可するものとします。

1. **お客様の明示的な同意を得たうえで、AnyDesk のリモート ID を取得すること。**

   各リモート接続の前に、当社はお客様の AnyDesk リモート ID を確認し、これが第三者に開示されていないことを確認します。

2. **お客様のコンピューター画面を遠隔操作し、関連する情報を閲覧または変更すること。**

   これには主として、コンピューターのネットワーク情報の確認、ルーター管理画面へのアクセスによる設定内容の確認、および問題の診断と修正のために必要なその他の操作が含まれます。確認の結果、問題の原因がルーター設定の誤りにあると判断した場合、当社はその問題を解決するためにルーター設定を変更することがあります。

3. **AnyDesk ソフトウェアによる情報処理を許可すること。**

   当社が AnyDesk を用いてサポートを提供した後、AnyDesk が操作ログを保持する場合があります。詳細については、[AnyDesk Privacy Policy](https://anydesk.com.cn/en/privacy){target="blank"} をご参照ください。
