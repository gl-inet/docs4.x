# Protokoll

Gehen Sie auf der linken Seite des Web-Admin-Panels zu **SYSTEM** -> **Log**.

Auf der Seite **Log** können Sie Protokolle von System, Kernel, Crash, Cloud und Nginx zur Analyse und Fehlerbehebung einsehen.

Einige Mobilfunkmodelle mit eSIM-Unterstützung stellen zusätzlich ein eSIM-Protokoll bereit.

Hier sehen Sie ein Beispiel für den GL-X3000.

![log page](https://static.gl-inet.com/docs/router/en/4/interface_guide/log/log.png){class="glboxshadow"}

Die **Filterfunktion** ist auf der Seite „System Log“ ab Firmware v4.8 verfügbar. Sie können nach Ebene oder Modul filtern oder Schlüsselwörter eingeben, um Protokolle schnell zu durchsuchen.

Klicken Sie auf **Refresh**, um die neuesten Protokollinformationen abzurufen.

Klicken Sie auf **Export Log**, um Protokollinformationen von System, Kernel, Crash und Cloud zu exportieren. Wenn Sie Feedback an GL.iNet senden, können Sie die exportierte Protokolldatei an den technischen Support von GL.iNet schicken, damit Probleme schneller analysiert werden können.
