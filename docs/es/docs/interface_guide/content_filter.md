# Content Filter

Content Filter proporciona seguridad inteligente en línea mediante clasificación basada en DPI, bloqueando automáticamente sitios web dañinos o maliciosos para mantener su red limpia y segura.

**Nota**: Esta función está disponible actualmente solo en **GL-MT5000 (Brume 3)**.

---

En el lado izquierdo del panel de administración web, vaya a **FLOW CONTROL** > **Content Filter**.

Personalice el contenido bloqueado, como aplicaciones, dominios o direcciones IP, y haga clic en **Apply**.

![content filter](https://static.gl-inet.com/docs/router/en/4/interface_guide/content_filter/content-filter.png){class="glboxshadow"}

- **Blocked Apps List**: Esta sección incluye tres categorías preseleccionadas: Gambling, Adult y Malware. Cuando está habilitada, se bloquearán los sitios web, servicios o aplicaciones relacionados con estas tres categorías. Puede hacer clic en Edit App para añadir otras categorías, por ejemplo, Game, Social Media o Shopping, según lo necesite.

- **Blocked Domain / IP List**: Esta sección le permite introducir manualmente dominios específicos, por ejemplo, google.com, o direcciones IP, por ejemplo, 192.168.8.0/24, para bloquear el acceso a ellos. La lista admite hasta 10000 entradas.
