# Dois-je configurer le WAN Ethernet lorsque j'utilise un VPN ?

La réponse est non. Lorsque vous configurez votre routeur GL.iNet comme client VPN, vous pouvez utiliser n'importe quelle source WAN. Votre LAN et votre WLAN seront couverts par le tunnel VPN.

## Configuration via le répéteur Wi-Fi

Vous pouvez connecter le routeur GL.iNet (configuré comme client VPN) à votre routeur principal sans fil ; de cette manière, le routeur GL.iNet fonctionnera comme un répéteur Wi-Fi.

![repater](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/repeater.jpg){class="glboxshadow"}

## Configuration via un câble Ethernet

Vous pouvez connecter le routeur GL.iNet (configuré comme client VPN) à votre routeur principal via un câble Ethernet.

![ethernet](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/ethernet.jpg){class="glboxshadow"}

## Configuration via le partage de connexion USB

Vous pouvez connecter le routeur GL.iNet (configuré comme client VPN) à votre téléphone via un câble USB ; cette méthode est également appelée USB Tethering.

![tethering](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/tethering.jpg){class="glboxshadow"}
