# Registro

En el lado izquierdo del panel web de administración, vaya a **SYSTEM** -> **Log**.

La página **Log** le permite ver los registros de System, Kernel, Crash, Cloud y Nginx para análisis y solución de problemas.

Algunos modelos celulares compatibles con eSIM también proporcionan registro eSIM.

Aquí se muestra un ejemplo del GL-X3000.

![log page](https://static.gl-inet.com/docs/router/en/4/interface_guide/log/log.png){class="glboxshadow"}

La función de **filtering** está disponible en la página **System Log** a partir del firmware v4.8. Puede elegir el nivel o el módulo, o introducir palabras clave para consultar rápidamente los registros.

Haga clic en **Refresh** para obtener la información de registro más reciente.

Haga clic en **Export Log** para exportar la información de registro de System, Kernel, Crash y Cloud. Cuando envíe comentarios a GL.iNet, puede enviar el archivo de registro exportado al soporte técnico de GL.iNet para un análisis más rápido del problema.
