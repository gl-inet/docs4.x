# Firmware zurücksetzen

Gehen Sie auf der linken Seite des Web-Admin-Panels zu **SYSTEM** -> **Reset Firmware**.

Wenn der Router nicht ordnungsgemäß funktioniert, können Sie versuchen, das Problem durch Zurücksetzen der Firmware zu beheben. Klicken Sie auf die Schaltfläche **Delete All and Reboot**, um die Firmware zurückzusetzen.

**Hinweis:** Alle aktuellen Einstellungen, Anwendungen und Daten gehen verloren. Der Vorgang dauert etwa 2 Minuten. Schalten Sie den Router während dieses Vorgangs **NICHT** aus.

![reset firmware](https://static.gl-inet.com/docs/router/en/4/tutorials/reset_firmware/reset_firmware_4.8.png){class="glboxshadow"}
