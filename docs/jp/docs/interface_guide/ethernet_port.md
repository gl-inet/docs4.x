# Ethernetポート

**注意**: このページはファームウェアv4.7では**ポート管理**として利用可能でしたが、ファームウェアv4.8.3で**Ethernetポート**に名前が変更されました。

---

Web管理パネルの左側で、**ネットワーク** -> **ポート管理**または**Ethernetポート**に移動します。

Ethernetポートの役割（WAN/LAN）を管理し、MACアドレスやネゴシ速度などのポート詳細情報を表示できます。

## WAN

このセクションには、ポートの役割（WANまたはLAN）、MACアドレス、ネゴシエート速度が表示されます。

![wan](https://static.gl-inet.com/docs/router/en/4/interface_guide/ethernet_port/wan.png){class="glboxshadow"}

- **WAN/LAN**: 物理WANポートの現在の動作モード。必要に応じてLANに設定できます。

- **MAC Mode**: デフォルトでファクトリーモード。クローンモードまたはランダムモードに切り替えることができます。

- **Mac Address**: WANインターフェースのMACアドレス。

- **Negotiated Network Port Rate**: WANインターフェースのネゴシエートされたリンク速度。有効なリンクが検出されたときにのみ表示されます。

## LAN

このセクションには、LANポートのネゴシエート速度が表示されます。有効なリンクが検出されたときにのみ表示されます。

![lan1](https://static.gl-inet.com/docs/router/en/4/interface_guide/ethernet_port/lan1.png){class="glboxshadow"}

一部のモデルでは、デュアルイーサネットWAN構成のためにLAN 1をWANポートとして切り替えることができます。以下の図のようにLANポートの役割を変更できます。詳細は[こちら](dual-ethernet_wan.md)をご覧ください。

![lan2](https://static.gl-inet.com/docs/router/en/4/interface_guide/ethernet_port/lan2.png){class="glboxshadow"}
