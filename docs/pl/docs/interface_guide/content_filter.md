# Filtr treści

Filtr treści zapewnia inteligentną ochronę online opartą na klasyfikacji DPI. Automatycznie blokuje szkodliwe lub złośliwe strony internetowe, pomagając utrzymać sieć w czystości i bezpieczeństwie.

**Uwaga**: Ta funkcja jest obecnie dostępna tylko na **GL-MT5000 (Brume 3)**.

---

Po lewej stronie panelu administracyjnego WWW przejdź do **FLOW CONTROL** > **Content Filter**.

Dostosuj blokowane treści (takie jak aplikacje, domeny lub adresy IP), a następnie kliknij **Apply**.

![content filter](https://static.gl-inet.com/docs/router/en/4/interface_guide/content_filter/content-filter.png){class="glboxshadow"}

- **Blocked Apps List**: Ta sekcja zawiera trzy wstępnie wybrane kategorie: Gambling, Adult i Malware. Po ich włączeniu blokowane będą wszystkie strony internetowe, usługi i aplikacje należące do tych kategorii. W razie potrzeby możesz kliknąć **Edit App**, aby dodać inne kategorie (np. Game, Social Media, Shopping).

- **Blocked Domain / IP List**: Ta sekcja pozwala ręcznie wprowadzić konkretne domeny (np. google.com) lub adresy IP (np. 192.168.8.0/24), aby zablokować do nich dostęp. Lista obsługuje maksymalnie 10000 wpisów.
