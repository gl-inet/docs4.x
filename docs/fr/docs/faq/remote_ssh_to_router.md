# Comment accéder à distance au terminal SSH d'un routeur GL.iNet ?

- **Méthode 1** : si le réseau de votre routeur dispose d'une [IP publique](../tutorials/how_to_check_if_isp_assigns_you_a_public_ip_address.md), configurez un [serveur WireGuard](../interface_guide/wireguard_server.md) et utilisez WireGuard pour vous connecter en SSH au routeur. Si vous n'avez pas d'IP publique, essayez les autres méthodes ci-dessous.
- **Méthode 2** : [liez votre appareil à GoodCloud](../interface_guide/cloud.md#setup), puis vous pourrez [vous connecter en SSH au routeur à distance](../interface_guide/cloud.md#remote-access-routers-terminal) en vous connectant au [site GoodCloud](https://www.goodcloud.xyz){target="_blank"}.
- **Méthode 3** : si vous souhaitez utiliser une solution SD-WAN, nous vous recommandons [AstroWarp](https://www.astrowarp.net){target="_blank"}.
- **Méthode 4** : si vous souhaitez utiliser une solution de proxy inverse, nous vous recommandons [AstroRelay](https://www.astrorelay.com){target="_blank"}.
