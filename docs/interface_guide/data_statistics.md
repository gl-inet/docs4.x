# データ統計

!!! note

    1. この機できるは現に **GL-MT5000 (Brume 3)** でのみ利用可できるです。

    2. この機できるはネットワークアクセラレーションと一緒に使用できません。有効にすると、正常に動作するためにネットワークアクセラレーションがから動のに無効になります。

データ統計は、アプリケーション別にネットワーク使用状況を分類して視覚化するインテリジェントなトラフィックインサイトダッシュボードを提供し、より良いネットワークのを握と制御のためにリアルタイムおよび履歴トラフィックを監視するのに役立ちます。

ルーターのWeb管理パネルにログインし、FLOW CONTROL > データ統計に移動して有効にします。「アプリケーションのデータ合計」を表示できます。

![data statistics](https://static.gl-inet.com/docs/router/en/4/interface_guide/data_statistics/data-statistics.png){class="glboxshadow"}

- **帯域幅使用量トップ10アプリ**：時間ベースのトレンドチャート（例：過去1日）を表示し、選択した期間におけるトップ10アプリケーションの帯域幅消費を示します。

    必要に応じて、タイムラインを過去1時間、過去1日、過去1週間で切り替えます。

- **アプリトラフィック統計**：ダウンロード、アップロード、合計帯域幅データなど、各アプリケーションの詳細なトラフィックメトリクスを表示します。

    必要に応じて、サーチバーで特定のアプリを検索します。
