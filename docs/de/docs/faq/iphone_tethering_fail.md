# iPhone-Tethering fehlgeschlagen

Wenn beim iPhone-Tethering ein Fehler auftritt, können Sie die folgenden Schritte versuchen, um das Problem zu beheben.

1. Stellen Sie sicher, dass **Maximise Compatibility** **aktiviert** ist (iPhone 8 oder neuer)

    ![maxcompat](https://static.gl-inet.com/docs/router/en/4/tutorials/iphone_tethering_fail/maxcompat.PNG){class="glboxshadow gl-50-desktop" }

2. Deaktivieren Sie den Hotspot auf Ihrem iPhone, warten Sie 5 Sekunden und aktivieren Sie ihn dann erneut.

    ![hotspot](https://static.gl-inet.com/docs/router/en/4/tutorials/iphone_tethering_fail/hotspot.PNG){class="glboxshadow gl-50-desktop"}

3. Stellen Sie sicher, dass iOS 15 oder höher verwendet wird.

4. Ziehen Sie das Lightning-/Type-C-Kabel ab und stecken Sie es erneut ein.

5. Verwenden Sie ein anderes USB-Kabel. Stellen Sie sicher, dass es die Datenübertragung unterstützt (nicht nur das Laden).

6. Verwenden Sie das Originalnetzteil des Routers.

7. Starten Sie sowohl den Router als auch Ihr iPhone neu. Manchmal behebt ein Neustart des iPhones das Tethering-Problem.
