# Come far funzionare Wi-Fi Calling su Opal (GL-SFT1200)?

## Aggiorna il firmware

Nel pannello di amministrazione web, vai su SYSTEM -> Upgrade e aggiorna il firmware all'ultima versione. Fai riferimento al [Firmware Download Center](https://dl.gl-inet.com/){target="_blank"} per verificare la versione più recente.

![opal4317](https://static.gl-inet.com/docs/router/en/4/faq/wifi_call/opal4317.jpg){class="glboxshadow"}

## Disabilita Network Acceleration

In base alla nostra esperienza e ai feedback dei clienti, Wi‑Fi Calling funziona correttamente dopo aver disabilitato Network/Hardware Acceleration.

Dopo l'aggiornamento del firmware, accedi al pannello di amministrazione web, vai su **NETWORK** -> **Network Acceleration**, quindi disattivala.

![networkacc](https://static.gl-inet.com/docs/router/en/4/faq/wifi_call/netacc.jpg){class="glboxshadow"}
