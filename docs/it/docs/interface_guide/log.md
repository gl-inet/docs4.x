# Log

Sul lato sinistro del pannello di amministrazione web, vai su **SYSTEM** -> **Log**.

La pagina Log consente di visualizzare i log di System, Kernel, Crash, Cloud e Nginx per analisi e risoluzione dei problemi.

Alcuni modelli cellulari che supportano eSIM forniscono anche il log eSIM.

Di seguito e' riportato un esempio con GL-X3000.

![log page](https://static.gl-inet.com/docs/router/en/4/interface_guide/log/log.png){class="glboxshadow"}

La funzione di **filtering** e' disponibile nella pagina System Log a partire dal firmware v4.8. Puoi scegliere il livello o il modulo, oppure inserire parole chiave per cercare rapidamente nei log.

Fai clic su **Refresh** per ottenere le informazioni di log piu' recenti.

Fai clic su **Export Log** per esportare le informazioni di log di System, Kernel, Crash e Cloud. Quando invii un feedback a GL.iNet, puoi inviare il file di log esportato al supporto tecnico GL.iNet per un'analisi del problema piu' rapida.
