# Wie lade ich die Firmware für meinen GL.iNet-Router herunter?

Wenn Sie Ihren Router aktualisieren möchten (oder ihn über U-Boot wiederherstellen möchten), laden Sie die Firmware-Dateien mit den folgenden Schritten herunter:

1. Wählen Sie im [GL.iNet Firmware Download Center](https://dl.gl-inet.com) in der Suchleiste Ihr Routermodell aus.
2. Wählen Sie oben einen Reiter aus:
    * **Stable:** Die offizielle Version für den täglichen Einsatz und für die meisten Benutzer empfohlen.
    * **Beta** (nur für bestimmte Routermodelle verfügbar): Die Vorabversion, die sich noch in der Entwicklung befindet.
    * **Snapshot** (nur für bestimmte Routermodelle verfügbar): Der tägliche Build mit den neuesten Aktualisierungen.
3. Klicken Sie auf die Schaltfläche **Download**.
