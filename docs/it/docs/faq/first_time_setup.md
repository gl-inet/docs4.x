# Configurazione iniziale

La configurazione iniziale dei router GL.iNet è molto simile. La maggior parte dei modelli dispone di un modulo Wi-Fi, mentre alcuni no.

Per questo motivo, la guida seguente è suddivisa in due casi in base alla presenza del modulo Wi-Fi.

* [Per i modelli dotati di Wi-Fi](#per-i-modelli-dotati-di-wi-fi)
* [Per i modelli senza Wi-Fi](#per-i-modelli-senza-wi-fi)

## Per i modelli dotati di Wi-Fi

I passaggi seguenti usano il GL-MT3000 (Beryl AX) come esempio.

Prepara i seguenti elementi inclusi nella confezione.

- Un router GL-MT3000 (Beryl AX)
- Un alimentatore
- Un cavo Ethernet

Guarda questa videoguida oppure segui i passaggi seguenti.

<iframe width="560" height="315" src="https://www.youtube.com/embed/WW8wGk68lEU" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

1. **Accensione**

    Collega un'estremità dell'alimentatore al router e l'altra a una presa di corrente. Il router si accenderà automaticamente.
    
    Alcuni modelli (come Slate AX) sono dotati di uno slot per scheda TF. Se devi usare una scheda TF, inseriscila prima di accendere il router. L'hot-swap delle schede TF non è supportato.

2. **Collegamento al router**

    Collega un dispositivo (ad esempio smartphone, laptop o computer) al router tramite cavo Ethernet oppure tramite Wi-Fi.

    * Collegamento tramite Ethernet: collega un dispositivo alla porta LAN del router tramite un cavo Ethernet.

    * Collegamento tramite Wi-Fi: trova l'SSID Wi-Fi e la Wi-Fi Key sull'etichetta inferiore del router. Cerca l'SSID del router sul dispositivo, quindi inserisci la Wi-Fi Key.

        !!! tip
        
            1. Il codice QR sull'etichetta inferiore contiene le informazioni Wi-Fi predefinite. Puoi collegarti rapidamente scansionandolo con un lettore di codici QR.
            2. Per alcuni modelli precedenti, se la Wi-Fi Key non è presente sull'etichetta, prova con "goodlife".
        
    Dopo esserti collegato al Wi-Fi, potresti non avere accesso immediato a Internet. Segui prima il passaggio successivo per configurare la rete.

3. **Accedi al pannello di amministrazione web**

    Apri un browser web (Chrome, Edge e Safari sono consigliati per una migliore compatibilità) e visita [http://192.168.8.1](http://192.168.8.1). Verrai indirizzato alla pagina di accesso GL.iNet. Se non riesci ad accedere al pannello di amministrazione web, fai clic [qui](cannot_access_web_admin_panel.md) per la risoluzione dei problemi.

    Imposta la password amministratore, quindi fai clic su **Next**.

    ![mt3000 set up admin password](https://static.gl-inet.com/docs/router/en/4/faq/first_time_setup/mt3000_admin_password.png){class="glboxshadow"}

    Configura il Wi-Fi. Se aggiorni le informazioni Wi-Fi, assicurati di ricollegarti alla rete aggiornata, quindi fai clic su **Next**

    ![mt3000 set up wifi](https://static.gl-inet.com/docs/router/en/4/faq/first_time_setup/mt3000_set_up_wifi.png){class="glboxshadow"}

    Attendi che il router completi l'inizializzazione.

    ![mt3000 initializing](https://static.gl-inet.com/docs/router/en/4/faq/first_time_setup/mt3000_initializing.png){class="glboxshadow"}

    Accederai quindi al pannello di amministrazione web del router.

    ![mt3000 admin panel](https://static.gl-inet.com/docs/router/en/4/faq/first_time_setup/mt3000_admin_panel.png){class="glboxshadow"}

2. **Collegamento al router**

    Collega un dispositivo (ad esempio laptop o computer) alla porta LAN del router tramite un cavo Ethernet.
