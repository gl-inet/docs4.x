# Où trouver le Device ID, l'adresse MAC et le numéro de série ?

## Méthode 1

Vous trouverez ces informations sous votre routeur.

Pour les modèles récents, l'étiquette inférieure ressemble à l'image ci-dessous et affiche les informations de base de l'appareil. Par exemple, dans l'image ci-dessous, l'adresse MAC est **E4:95:6E:40:DB:A1**, le Device ID est **hh0dba1** et le numéro de série est **7c0be4bb45d96057**.

![label of gl-x750](https://static.gl-inet.com/docs/router/en/4/tutorials/where_to_find_the_device_id_mac_sn/back_label_new.png){class="glboxshadow"}

Pour les anciens modèles, le Device ID n'apparaît pas sur l'étiquette inférieure, mais vous pouvez y trouver le DDNS, dont les 7 premiers caractères correspondent au Device ID. Prenons l'image ci-dessous comme exemple : son adresse MAC est **E4:95:6E:40:DB:A1**, son Device ID est **hh0dba1** et son numéro de série est **7c0be4bb45d96057**.

![label of gl-ar150](https://static.gl-inet.com/docs/router/en/4/tutorials/where_to_find_the_device_id_mac_sn/back_label_old.png){class="glboxshadow"}

## Méthode 2

Connectez-vous au panneau d'administration web, allez dans SYSTEM -> Overview, et vous trouverez les détails de votre appareil.

![device overview](https://static.gl-inet.com/docs/router/en/4/tutorials/where_to_find_the_device_id_mac_sn/overview_device_info.png){class="glboxshadow"}
