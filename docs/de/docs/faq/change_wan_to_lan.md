# WAN zu LAN ändern

Sie können den WAN-Port Ihres Routers in einen LAN-Port umwandeln. Das ist besonders nützlich, wenn Sie den Router im Repeater-Modus verwenden und der WAN-Port nicht benötigt wird. Durch die Umstellung von WAN auf LAN erhalten Sie einen zusätzlichen LAN-Port für erweiterte Verbindungen.

Führen Sie die folgenden Schritte aus, um WAN zu LAN zu ändern.

## Für Firmware 4.7 und höher

1. Lassen Sie den WAN-Port unverbunden.

2. Verbinden Sie ein Gerät mit dem Router und öffnen Sie das Web-Admin-Panel des Routers.

3. Navigieren Sie im Web-Admin-Panel zum Abschnitt **INTERNET** -> **Ethernet** und klicken Sie oben rechts auf das Zahnradsymbol.

![port management](https://static.gl-inet.com/docs/router/en/4/faq/change_wan_to_lan/ethernet_gear_icon.png){class="glboxshadow"}

Sie werden zur Seite **Port Management** weitergeleitet, auf der der Status des WAN-Ports als für WAN verwendet angezeigt wird.

![port management](https://static.gl-inet.com/docs/router/en/4/faq/change_wan_to_lan/port_management.jpg){class="glboxshadow"}

4. Klicken Sie auf **LAN**, um die Eigenschaften des Ethernet-Ports zu ändern, und klicken Sie anschließend auf **Apply**.

![switch to lan apply](https://static.gl-inet.com/docs/router/en/4/faq/change_wan_to_lan/switch_to_lan_apply.jpg){class="glboxshadow"}

Klicken Sie im eingeblendeten Warnfenster zur Bestätigung auf **Apply**.

![caution](https://static.gl-inet.com/docs/router/en/4/faq/change_wan_to_lan/caution.png){class="glboxshadow"}

**Hinweis**: Während dieses Vorgangs kann die Wi-Fi-Verbindung vorübergehend getrennt werden. Verbinden Sie sich nach Abschluss bitte erneut mit dem Router.

5. Zurück im Abschnitt Ethernet wird angezeigt, dass der WAN-Port jetzt als LAN-Port verwendet wird.

![wan using as lan](https://static.gl-inet.com/docs/router/en/4/faq/change_wan_to_lan/wan_using_as_lan.png){class="glboxshadow"}

Sie können ihn auf der Seite **Port Management** wieder auf WAN zurückstellen oder die RESET-Taste 4 Sekunden lang drücken, um die WAN-Schnittstelle neu zu starten.

## Für Firmware 4.6 und früher

1. Lassen Sie den WAN-Port unverbunden.

2. Verbinden Sie ein Gerät mit dem Router und öffnen Sie das Web-Admin-Panel.

3. Navigieren Sie im Web-Admin-Panel zum Abschnitt **INTERNET** -> **Ethernet**. Dort wird der Status des WAN-Ports als **Using as WAN** angezeigt. Klicken Sie auf **Change to LAN**.

![internet page](https://static.gl-inet.com/docs/router/en/4/tutorials/change_wan_to_lan/ethernet_no_cable.png){class="glboxshadow"}

4. Klicken Sie zur Bestätigung auf **Apply**.

![caution change wan as lan](https://static.gl-inet.com/docs/router/en/4/tutorials/change_wan_to_lan/ethernet_change_to_lan_caution.png){class="glboxshadow"}

**Hinweis**: Während dieses Vorgangs kann die Wi-Fi-Verbindung vorübergehend getrennt werden. Verbinden Sie sich nach Abschluss bitte erneut mit dem Router.

5. Zurück im Abschnitt Ethernet wird `Using as LAN` angezeigt.

![using as lan](https://static.gl-inet.com/docs/router/en/4/tutorials/change_wan_to_lan/ethernet_using_as_lan.png){class="glboxshadow"}

Sie können die Einstellung einfach rückgängig machen, indem Sie die obigen Schritte erneut ausführen.
