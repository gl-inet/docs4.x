# ディスプレイ管理

**Note**: この機能は **GL-BE3600 (Slate 7)** と **GL-E5800 (Mudi 7)** でのみ利用できます。

Web管理パネルの左側で、**SYSTEM** -> **Display Management** に移動します。

このページでは、タッチスクリーンディスプレイと関連設定を管理できます。

## GL-BE3600

![display management](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_display.png){class="glboxshadow"}

- **Function Management**: タッチスクリーンに表示する機能を管理します。

    ![function](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_function.png){class="glboxshadow"}

- **Lock Screen**: ウェイク表示や壁紙を含むロック画面の設定をカスタマイズします。

    ![lock screen](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_lockscreen.png){class="glboxshadow"}

    壁紙を **Customize** に切り替えると、5MB を超えない画像をアップロードできます。

    ![wallpaper customize](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_wallpaper_custom.png){class="glboxshadow"}

- **Brightness**: タッチスクリーンの明るさを調整します。スライダーを使うか、特定のレベル（1〜10）を入力して、周囲の明るさに合わせられます。
- **Auto Lock**: 操作がない場合に画面が自動ロックされるまでの時間を設定します。範囲は 1〜30 分です。
- **Screen Always On**: このオプションを切り替えて、タッチスクリーンを常時点灯させるか、無操作後に消灯させるかを選択します。
- **Enable Screen Passcode**: タッチスクリーンにパスコード（4桁の数字のみ）を設定し、追加のセキュリティを提供します。

    ![screen passcode](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_passcode.png){class="glboxshadow"}

## GL-E5800

![display management](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/e5800_display.png){class="glboxshadow"}

- **Wallpaper**: 壁紙とウェイク表示スタイルをカスタマイズします。
- **Brightness**: タッチスクリーンの明るさを調整します。スライダーを使うか、割合を入力して、周囲の明るさに合わせられます。
- **Personalised Signature**: タッチスクリーンにカスタムテキストを追加して、見た目の個性付けや識別に利用できます。
- **Auto Lock**: 操作がない場合に画面が自動ロックされるまでの時間を設定します。範囲は 15 秒〜5 分です。
- **Passcode**: タッチスクリーンにパスコードを設定して、追加のセキュリティを提供します。
