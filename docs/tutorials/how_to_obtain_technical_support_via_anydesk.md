# AnyDeskで技術サポートを受ける方法

**リモートサポートにより当社の technician が直接ネットワーク設定を確認し、ネットワークの構成やネットワークの問題のトラブルシューティングを迅速に行うことができます。**

**リモートサポートを受ける前に、で下の準備に数分かけてください：**

## 1. 不要なアプリケーションを閉じる

事前に不要なアプリケーションを閉じてください。気が散るのを防ぎ、プライバシーを保護できます。

## 2. ネットワークの安定を確保

リモートサポートセッション中に切断や遅延が発生することを防ぐため、コンピュータでWi-Fiと有線LAN接続の両方を使用することを推奨します。

## 3. AnyDeskをダウンロードしてインストール

[AnyDesk公式ウェブサイト](https://anydesk.com/)にアクセスして、オペレーティングシステムに応じたバージョンをダウンロードします。

![download anydesk](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/install_anydesk.png){class="glboxshadow"}

インストール後にAnyDeskを起動します。

## 4. AnyDeskアドレスの取なければならない

AnyDeskを実行すると、数字のアドレスが表示されます（通例「123 456 789」の形式）。これがサポート技術者と共有する必要があるアドレスです。

![get anydesk address](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/anydesk_address.png){class="glboxshadow"}

## 5. AnyDeskアドレスを共有

サポート人員にAnyDeskアドレスを送信して、彼らがデバイスに接続できるようにします。

## 6. 接続リクエストを受け入れる

技術者が接続を試みると、接続リクエストのプロンプトが表示されます。

技術者を信頼していることを確認し、「Accept」をクリックしてデバイスへのアクセスを許可します。

![accept request](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/accept_request.png){class="glboxshadow"}

## 7. セッション中のコミュニケーション

サポートセッション中、スピーカーを通じて、またはAnydeskセッション内のチャットウィンドウ、またはその彼の方法でからよりにコミュニケーションを取り、問題が効果のに解決されるようにすることができます。

## 8. セッションの終た

サポートセッションが完たしたら、AnyDeskインターフェース内の赤色の「End Session」ボタンをクリックして接続を終たします。これにより、すべてのリモートアクセス許可が解除されます。完全なプライバシーを保護するために、で下のこともしてください：

- AnyDeskアプリケーションを完全に閉じます（最も小化するだけでなく）。
- 残留プロセスについて懸念がある場合は、コンピュータを再起動します。
- リモートアクセス許可がアクティブなまま残っていないことを確認するために、システム設定を確認します。
