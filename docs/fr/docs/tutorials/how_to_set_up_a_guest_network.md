# Comment configurer un réseau Wi-Fi invité sur les routeurs GL.iNet

Ce tutoriel vous montre comment configurer un réseau Wi-Fi invité sur les routeurs GL.iNet. 

Un réseau Wi-Fi invité est un réseau distinct sur un routeur pour les appareils que vous souhaitez isoler de vos appareils principaux. Cela inclut les appareils des invités ainsi que vos appareils domotiques. En créant un réseau invité, vous pouvez empêcher les accès non autorisés et les attaques potentielles contre votre réseau principal, ce qui renforce la sécurité de votre réseau. 

Il existe deux méthodes pour configurer un réseau Wi-Fi invité sur votre routeur GL.iNet : via le [panneau d’administration du routeur](#set-up-a-guest-network-via-the-admin-panel) ou via l’[application mobile GL.iNet](#set-up-a-guest-network-via-the-glinet-mobile-app). 

## Configurer un réseau invité via le panneau d’administration {#set-up-a-guest-network-via-the-admin-panel}

### 1. Activer le réseau Wi-Fi invité

1. Dans un navigateur web, saisissez `192.168.8.1`. 
2. Saisissez votre mot de passe, puis cliquez sur **Login**. 
3. Dans la barre latérale gauche, cliquez sur **Wireless**.
![click wireless](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/click-wireless.jpeg){class="glboxshadow"}
4. Dans la section de votre réseau Wi-Fi 5 GHz ou 2.4 GHz, cliquez sur l’onglet **Guest Wi-Fi**. 
![click guest](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/click-guest-wifi.jpeg){class="glboxshadow"}
5. À côté de **Enable Wi-Fi**, activez l’interrupteur. 

### 2. Définir un nouveau mot de passe pour le réseau Wi-Fi invité

Pour renforcer votre sécurité, il est recommandé de définir un nouveau mot de passe pour votre réseau Wi-Fi invité. Suivez ces étapes : 

1. Cliquez sur **Modify**.
![click modify](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/click-modify.jpeg){class="glboxshadow"}
2. À côté de **Wi-Fi Password**, saisissez un nouveau mot de passe. (Vous pouvez également saisir un nouveau nom de réseau à côté de **Wi-Fi Name**.)
3. Ensuite, cliquez sur **Apply**.

Vous avez maintenant configuré un réseau Wi-Fi invité. Pour y connecter vos appareils, repérez le nom du réseau invité dans la liste des réseaux disponibles et saisissez le mot de passe.  

## Configurer un réseau invité via l’application mobile GL.iNet {#set-up-a-guest-network-via-the-glinet-mobile-app}

**Note:** Avant de commencer, installez et configurez l’application mobile GL.iNet sur votre appareil. 

### 1. Activer le réseau Wi-Fi invité

1. Sur l’écran principal de l’application, appuyez sur l’onglet **Network** > **Wireless**.
2. Repérez **Guest Network**. 
3. Pour le réseau sur lequel vous souhaitez configurer un réseau Wi-Fi invité, activez l’interrupteur. 
![toggle the switch](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/toggle-on-guest-network.jpeg){class="glboxshadow"}

### 2. Définir un nouveau mot de passe pour le réseau Wi-Fi invité

Pour renforcer votre sécurité, il est recommandé de définir un nouveau mot de passe pour votre réseau Wi-Fi invité. Suivez ces étapes : 

1. Appuyez sur l’option **SSID** du réseau sur lequel vous avez configuré le réseau Wi-Fi invité. 
2. À côté de **Password**, saisissez un nouveau mot de passe. (Vous pouvez également saisir un nouveau nom de réseau dans le champ SSID.) 
![enter a new password](https://static.gl-inet.com/docs/router/en/4/tutorials/how_to_set_up_a_guest_network/tap-password.jpeg){class="glboxshadow"}
3. Appuyez sur **Done**.

Vous avez maintenant configuré un réseau Wi-Fi invité. Pour y connecter vos appareils, repérez le nom du réseau invité dans la liste des réseaux disponibles et saisissez le mot de passe.
