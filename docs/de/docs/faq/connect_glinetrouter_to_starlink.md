# So verbinden Sie einen GL.iNet Router mit einer Starlink-Schüssel

Wenn Sie einen GL.iNet Router mit einer Starlink-Schüssel verbinden möchten, folgen Sie den untenstehenden Schritten.

**Topologie**

![starlinkgl](https://static.gl-inet.com/docs/router/en/4/faq/starlink/starlinkgl.jpg){class="glboxshadow"}

1. Schließen Sie die Stromversorgung Ihres Starlink-Routers an die Starlink-Schüssel an.

2. Verbinden Sie den LAN-Port des Starlink-Routers mit dem WAN-Port des GL.iNet Routers.

3. Verbinden Sie ein Gerät mit dem LAN-Port oder dem WLAN des GL.iNet Routers und konfigurieren Sie anschließend den Router für den Internetzugang. [Ersteinrichtung](first_time_setup.md)

    **Hinweis**: Wenn Ihr GL.iNet Router [Network Acceleration](../interface_guide/network_acceleration.md) unterstützt, gehen Sie bitte im Web-Admin-Panel zu **NETWORK** -> **Network Acceleration** und deaktivieren Sie diese Funktion.

    ![netacc](https://static.gl-inet.com/docs/router/en/4/faq/starlink/netacc.jpg){class="glboxshadow"}

4. Öffnen Sie die Starlink App, gehen Sie zu Settings und aktivieren Sie den **Bypass mode**.
