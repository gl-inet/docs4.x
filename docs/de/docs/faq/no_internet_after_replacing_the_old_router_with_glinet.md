# Was soll ich tun, wenn ich nach dem Ersetzen des alten Routers durch einen GL.iNet-Router kein Internet habe?

Wenn Sie nach dem Ersetzen Ihres alten Routers durch einen GL.iNet-Router nicht auf das Internet zugreifen können, erhält Ihr neuer Router möglicherweise keine IP-Adresse (über DHCP oder PPPoE), wenn sein WAN-Port mit dem Modem verbunden ist.

Möglicherweise hat Ihr Internetanbieter eine **MAC-Bindung** aktiviert, sodass nur die MAC-Adresse Ihres vorherigen Routers eine Verbindung zum Netzwerk herstellen darf.

Versuchen Sie die folgenden Methoden, um dieses Problem zu beheben.

- Klonen Sie die MAC-Adresse Ihres alten Routers auf Ihren neuen GL.iNet-Router.
- Starten Sie Ihr Modem neu. Wenn das Problem weiterhin besteht, schalten Sie es für 30 Minuten aus und dann wieder ein.
- Kontaktieren Sie Ihren Internetanbieter und bitten Sie ihn, das Modem zurückzusetzen.
