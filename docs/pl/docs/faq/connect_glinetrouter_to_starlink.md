# Jak podłączyć router GL.iNet do anteny Starlink?

Jeśli chcesz podłączyć router GL.iNet do anteny Starlink, wykonaj poniższe kroki.

**Topologia**

![starlinkgl](https://static.gl-inet.com/docs/router/en/4/faq/starlink/starlinkgl.jpg){class="glboxshadow"}

1. Podłącz zasilanie routera Starlink do anteny Starlink.

2. Podłącz port LAN routera Starlink do portu WAN routera GL.iNet.

3. Podłącz urządzenie do portu LAN routera GL.iNet lub do jego Wi-Fi, a następnie skonfiguruj router pod kątem dostępu do Internetu. [Pierwsza konfiguracja](first_time_setup.md)

    **Uwaga**: Jeśli router GL.iNet obsługuje funkcję [Network Acceleration](../interface_guide/network_acceleration.md), przejdź w panelu administracyjnym WWW do **NETWORK** -> **Network Acceleration** i wyłącz tę funkcję.

    ![netacc](https://static.gl-inet.com/docs/router/en/4/faq/starlink/netacc.jpg){class="glboxshadow"}

4. Otwórz aplikację Starlink, przejdź do **Settings** i włącz **Bypass mode**.
