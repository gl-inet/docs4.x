# IGMP Snooping

Dans le panneau d'administration web, accédez à **NETWORK** -> **IGMP Snooping**.

Vous pouvez activer IGMP Snooping pour utiliser la fonction multicast sur le routeur.

IGMP Snooping écoute les paquets du protocole IGMP, en extrait les informations correspondantes, établit et maintient les entrées de transfert multicast de couche 2, puis transmet les données du groupe multicast à l'hôte qui a rejoint ce groupe. Les autres hôtes ne peuvent pas recevoir les données de ce groupe multicast.

**Note**: IGMPv3 est compatible avec les versions v1 et v2. Utilisez v3 par défaut et changez de version seulement si vous rencontrez un problème.
