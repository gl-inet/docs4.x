# IGMP Snooping

Gehen Sie auf der linken Seite des Web-Admin-Panels zu **NETWORK** -> **IGMP Snooping**.

Sie können IGMP Snooping aktivieren, um die Multicast-Funktion auf dem Router zu verwenden.

IGMP Snooping überwacht IGMP-Protokollpakete, extrahiert die entsprechenden Informationen, erstellt und pflegt die Weiterleitungsbeziehungen für Multicast auf Layer 2 und leitet dann die Multicast-Gruppendaten an Hosts weiter, die der Multicast-Gruppe beigetreten sind, während andere Hosts diese Daten nicht empfangen können.

**Hinweis**: IGMPv3 ist mit v1 und v2 kompatibel. Verwenden Sie standardmäßig v3 und wechseln Sie nur, wenn Probleme auftreten.
