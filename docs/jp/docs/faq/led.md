## セルラールーター

### GL-XE3000

![gl-xe3000 led](https://static.gl-inet.com/docs/router/en/4/user_guide/gl-xe3000/hardware_info/xe3000led.jpg){class="glboxshadow gl-70-desktop"}

* 電源LED
	* 緑: 電源オン
	* オフ: 電源オフ

* バッテリーLED（充電中）
	* 赤が点滅: バッテリー残量 0%-20%
	* 紫が点滅: バッテリー残量 21%-50%
	* 緑が点滅: バッテリー残量 51%-99%
	* 緑で点灯: バッテリー残量 100%

* バッテリーLED（非充電時）

	非充電時、電源ボタンを押すとバッテリーLEDが5秒間点灯します。

	* 赤: バッテリー残量 0%-20%
	* 紫: バッテリー残量 21%-50%
	* 緑: バッテリー残量 51%-100%
 	
* インターネットLED
	* オン: インターネットに接続
	* オフ: インターネットが切断

* 2.4G Wi-Fi LED
	* オン: 2.4G Wi-Fiが準備完了
	* 点滅: 2.4G Wi-Fiが通信中

* 5G Wi-Fi LED
	* オン: 5G Wi-Fiが準備完了
	* 点滅: 5G Wi-Fiが通信中

* セルラーLED
	* 携帯基地局に接続済み

## セキュリティゲートウェイ＆ミニルーター

### GL-MT5000

![gl-mt5000 interface](https://static.gl-inet.com/docs/router/en/4/user_guide/gl-mt5000/hardware_info/mt5000_interface.png){class="glboxshadow"}

* システムLED
	* 青: システム起動中
	* 青が点滅: ネットワーク検索中
	* 白: 正常に動作中
* VPN LED
	* オフ: VPNクライアントが未接続
	* 点滅: VPNクライアントが接続中
	* 点灯: VPNクライアントが接続済み
	
注意: VPNサーバーのステータスはLEDに表示されません。

### GL-MT300N-V2

![gl-mt300n-v2 interface](https://static.gl-inet.com/docs/router/en/4/user_guide/gl-mt300n-v2/product_info/mt300n-v2_interface.png){class="glboxshadow"}

- 電源LED
	- オン: 電源オン
	- オフ: 電源オフ

- 中央LED（設定可能）
	- オン: このLEDはカスタマイズ済み
	- オフ: このLEDはカスタマイズされていません（デフォルトモード）

- 無線LED
	- オン: 2.4G Wi-Fiが接続
	- オフ: 2.4G Wi-Fiが切断

	初期ロットでは、通常の動作状態で無線LEDが白の代わりに赤で表示される場合があります。
