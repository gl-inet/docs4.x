# Ethernetポート

**注意**: このページはファームウェアv4.7では**ポート管理**として利用可能でしたが、ファームウェアv4.8.3で**Ethernetポート**に名前が変更されました。

---

Web管理パネルの左側で、**ネットワーク** -> **ポート管理**または**Ethernetポート**に移動します。

Ethernetポートのプロパティを管理したり、ポート仕様を表示したりできます。

## WAN

このセクションには、ポートの役割（WANまたはLANとして使用）、MACモードとMACアドレス、ネゴシエートされたネットワークポート速度が表示されます。

![wan](https://static.gl-inet.com/docs/router/en/4/interface_guide/ethernet_port/wan.png){class="glboxshadow"}

- WAN/LAN: 物理WANポートの現在のステータス。必要に応じてLANポートとして使用できます。

- MAC Mode: WANポートのMACモードはデフォルトでファクトリー設定です。クローンモードまたはランダムモードに切り替えることができます。

- Mac Address: WANインターフェースのMACアドレス。

- Negotiated Network Port Rate: WANインターフェースの現在のネゴシエート速度。

## LAN

このセクションには、ポートの役割とネゴシエートされたネットワークポート速度が表示されます。

![lan](https://static.gl-inet.com/docs/router/en/4/interface_guide/ethernet_port/lan.png){class="glboxshadow"}

## デュアルイーサネットWAN

一部のモデルは、デュアルイーサネットWANシナリオのためにLAN1ポートをWANポートに切り替えをサポートします。詳細については[こちら](dual-ethernet_wan.md)をご覧ください。
