# Échec du partage de connexion de l'iPhone

Si vous rencontrez un échec du partage de connexion avec un iPhone, essayez les étapes suivantes pour résoudre le problème.

1. Assurez-vous que **Maximise Compatibility** est **Enabled** (iPhone 8 ou version ultérieure).

    ![maxcompat](https://static.gl-inet.com/docs/router/en/4/tutorials/iphone_tethering_fail/maxcompat.PNG){class="glboxshadow gl-50-desktop" }

2. Désactivez le point d'accès sur votre téléphone, attendez 5 secondes, puis réactivez-le.

    ![hotspot](https://static.gl-inet.com/docs/router/en/4/tutorials/iphone_tethering_fail/hotspot.PNG){class="glboxshadow gl-50-desktop"}

3. Assurez-vous que la version d'iOS est 15 ou supérieure.

4. Débranchez puis rebranchez le câble Lightning / Type-C.

5. Utilisez un autre câble USB. Assurez-vous qu'il prend en charge le transfert de données (et pas uniquement la charge).

6. Utilisez l'alimentation d'origine du routeur.

7. Redémarrez à la fois le routeur et votre iPhone. Dans certains cas, le redémarrage de l'iPhone résout le problème de partage de connexion.
