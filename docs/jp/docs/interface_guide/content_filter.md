# コンテンツフィルター

コンテンツフィルターは、DPI ベースの分類によるスマートなオンライン保護を提供し、有害または悪意のある Web サイトを自動的にブロックして、ネットワークを安全でクリーンな状態に保ちます。

**Note**: この機能は現在 **GL-MT5000 (Brume 3)** でのみ利用できます。

---

Web Admin Panel の左側で、**FLOW CONTROL** > **Content Filter** に移動します。

ブロックするコンテンツ（アプリ、ドメイン、IP アドレスなど）を設定し、**Apply** をクリックします。

![content filter](https://static.gl-inet.com/docs/router/en/4/interface_guide/content_filter/content-filter.png){class="glboxshadow"}

- **Blocked Apps List**: このセクションには、Gambling、Adult、Malware の 3 つのカテゴリがあらかじめ選択されています。有効にすると、これら 3 つのカテゴリに関連する Web サイト、サービス、アプリがブロックされます。必要に応じて **Edit App** をクリックし、Game、Social Media、Shopping などのカテゴリを追加できます。

- **Blocked Domain / IP List**: このセクションでは、特定のドメイン（例: google.com）や IP アドレス（例: 192.168.8.0/24）を手動で入力してブロックできます。リストは最大 10000 件の項目に対応しています。
