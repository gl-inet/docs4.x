# Dual-Ethernet WAN

La funzione Dual-Ethernet WAN consente di trasformare la LAN1 predefinita in una seconda porta WAN per l'accesso dual Ethernet, offrendo una connettivita' di backup affidabile e supportando l'aggregazione della banda, dove compatibile, per scenari ad alto carico. Consente inoltre di collegarsi contemporaneamente a due reti separate, ad esempio lavoro e rete personale, aumentando la flessibilita' senza hardware aggiuntivo.

    **Nota**: GL-E5800 (Mudi 7) e' dotato di una porta Ethernet, LAN per impostazione predefinita ma commutabile in WAN, e di una **porta USB-C con supporto OTG**. Per aggiungere una seconda porta Ethernet per Dual-Ethernet WAN, collega alla porta USB-C un adattatore USB-C-Ethernet venduto separatamente.

## Configurazione

1. Accedi al pannello di amministrazione web del router.

2. Vai su **NETWORK** -> **Port Management** -> scheda **LAN**, cambia la funzione della porta in WAN e fai clic su **Apply**.

    ![dual ethernet wan 1](https://static.gl-inet.com/docs/router/en/4/tutorials/dual-ethernet_wan/dual_ethernet_1.png){class="glboxshadow"}

3. Nella finestra pop-up, fai clic su **Apply**.

    ![dual ethernet wan 2](https://static.gl-inet.com/docs/router/en/4/tutorials/dual-ethernet_wan/dual_ethernet_2.png){class="glboxshadow"}

4. Dopo aver abilitato Dual-Ethernet WAN, puoi configurare qui le funzioni multi-wan [qui](../interface_guide/multi-wan.md).
