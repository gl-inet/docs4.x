# Cosa devo fare se non ho Internet dopo aver sostituito il vecchio router con un router GL.iNet

Se non riesci ad accedere a Internet dopo aver sostituito il vecchio router con un router GL.iNet, il nuovo router potrebbe non riuscire a ottenere un indirizzo IP (tramite DHCP o PPPoE) quando la porta WAN è collegata al modem.

Il tuo Internet Service Provider potrebbe avere abilitato il **MAC binding**, che consente solo all'indirizzo MAC del router precedente di collegarsi alla rete.

Prova i seguenti metodi per risolvere il problema.

- Clona sul nuovo router GL.iNet l'indirizzo MAC del vecchio router.
- Riavvia il modem. Se il problema persiste, spegnilo per 30 minuti e poi riaccendilo.
- Contatta il tuo Internet Service Provider per reimpostare il modem.
