# セルラールーターの外部アンテナを取り付けまたは交換する方法

このチュートリアルでは、一部のセルラールーター向けに外部アンテナを取り付ける方法と交換する方法を紹介します。

## 外部アンテナを取り付ける

### Spitz AX / Puli AX

Spitz AX (GL-X3000) と Puli AX (GL-XE3000) には、次の外部アンテナが付属しています。

* 2 x Wi-Fi antennas (RP-SMA, 2.4G~2.5GHz, 5.15~5.84GHz)
* 4 x 5G NR antennas (SMA, 600MHz~6GHz)

外部アンテナを取り付けるには、次の手順に従ってください。

1. アンテナを取り付けます（順不同）。

    * 同じ Wi-Fi 6 アンテナ 2 本を **Wi-Fi 1** と **Wi-Fi 2** ポートに取り付けます。
    * 同じ 5G NR アンテナ 4 本を **MAIN**、**DIV**、**MIMO 1**、**MIMO2** に取り付けます。

    ![x3000 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x3000-antennas.jpg){class="gl-80-desktop glboxshadow"}

2. 最適な性能を得るため、アンテナは上向きに配置してください。

### Spitz Plus

Spitz Plus (GL-X2000) には、次の外部アンテナが付属しています。

* 2 x Wi-Fi antennas (RP-SMA, 2.4G~2.5GHz, 5.15~5.84GHz)
* 2 x 4G antennas (SMA, 600MHz~3.8GHz)

外部アンテナを取り付けるには、次の手順に従ってください。

1. アンテナを取り付けます（順不同）。

    - 同じ Wi-Fi アンテナ 2 本を **Wi-Fi 6** ポートに取り付けます。
    - 同じ 4G アンテナ 2 本を **4G** ポートに取り付けます。

    ![x2000 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x2000-back.png){class="gl-80-desktop glboxshadow"}

2. 最適な性能を得るため、アンテナは上向きに配置してください。

### Spitz

Spitz (GL-X750/GL-X750V2) には、外部 4G アンテナ 2 本 (SMA, 700M~2700MHz) が付属しています。

外部アンテナを取り付けるには、次の手順に従ってください。

1. 同じ 4G アンテナ 2 本をアンテナポートに取り付けます。

    ![x750v2 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x750v2_back.png){class="gl-80-desktop glboxshadow"}

2. 最適な性能を得るため、アンテナは上向きに配置してください。

### Collie

Collie (GL-X300B-RS485) には、次のアンテナが付属しています。

* 1 x External 4G antenna
* 2 x External Wi-Fi antennas

アンテナを取り付けるには、次の手順に従ってください。

1. アンテナを取り付けます（順不同）。

    - 同じ Wi-Fi アンテナ 2 本を **2.4G** ポートに取り付けます。
    - 4G アンテナ 1 本を **4G** ポートに取り付けます。

    ![x300b antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x300b-rs485.png){class="gl-80-desktop glboxshadow"}

    ![x300b antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x300b-rs485-back.png){class="gl-80-desktop glboxshadow"}

2. 最適な性能を得るため、アンテナは上向きに配置してください。

## 外部アンテナを交換する

以下の案内は、Mudi 7 (GL-E5800) を除く、外部アンテナ端子を備えたすべてのセルラールーターに適用されます。

追加の Wi-Fi アンテナや 4G/5G NR アンテナを購入して、パッケージに含まれているアンテナと交換し、ネットワーク構成をカスタマイズできます。

ただし、最適な性能と互換性を確保するため、出荷前に十分なテストを行っている純正アンテナの使用を推奨します。

サードパーティ製アンテナを使用する場合は、正しいコネクタタイプのアンテナを選べるよう、以下のセクションを参照してください。

**Tips**: 誤ったアンテナを購入した場合は、コネクタを変換するアダプター（例: SMA-RP male to SMA female）の購入を検討してください。

### Wi-Fiアンテナ

Wi-Fi アンテナの基部には **RP-SMA female connector**（外ネジ・内ピン）が使われています。そのため、**RP-SMA male connector**（内ネジ・内穴）を備えた Wi-Fi アンテナが必要です。

![rp-sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/rp-sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### 5G NR antennas

5G NR アンテナの基部には **SMA female connector**（外ネジ・内穴）が使われています。そのため、**SMA male connector**（内ネジ・内ピン）を備えた 5G NR アンテナが必要です。

![sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/sma-connectors.jpg){class="gl-80-desktop glboxshadow"}
