# IGMP Snooping

Po lewej stronie panelu administracyjnego WWW przejdź do **NETWORK** -> **IGMP Snooping**.

Możesz włączyć IGMP snooping, aby korzystać z funkcji multicast na routerze.

IGMP Snooping nasłuchuje pakietów protokołu IGMP, wyodrębnia odpowiednie informacje, tworzy i utrzymuje tablicę przekazywania multicastu warstwy 2, a następnie przekazuje dane grupy multicast do hosta, który do niej dołączył, podczas gdy pozostałe hosty nie mogą odbierać danych tej grupy multicast.

**Uwaga**: IGMPv3 jest zgodny z v1 i v2. Domyślnie używaj v3 i zmień wersję tylko wtedy, gdy wystąpi problem.
