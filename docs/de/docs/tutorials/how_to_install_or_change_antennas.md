# So installieren oder wechseln Sie externe Antennen für Spitz AX oder Puli AX

Wenn Sie Ihren Spitz AX (GL-X3000) oder Puli AX (GL-XE3000) zum ersten Mal einrichten, müssen Sie die sechs externen Antennen installieren, die im Lieferumfang enthalten sind. Diese externen Antennen sind:

* 2 x Wi-Fi-Antennen (mit der Kennzeichnung "Wi-Fi 6")
* 4 x 5G-NR-Antennen (mit der Kennzeichnung "5G NR")

In den folgenden Abschnitten erfahren Sie, wie Sie die Antennen installieren oder austauschen.

## Externe Antennen installieren

Gehen Sie zum Installieren der Antennen wie folgt vor:

1. Befestigen Sie die Antennen (in beliebiger Reihenfolge):
    * Befestigen Sie die beiden identischen Wi-Fi-6-Antennen an den Ports **Wi-Fi 1** und **Wi-Fi 2** (eine bestimmte Zuordnung ist nicht erforderlich).
    * Befestigen Sie die vier identischen 5G-NR-Antennen an den Ports **MAIN**, **DIV**, **MIMO 1** und **MIMO2** (eine bestimmte Zuordnung ist nicht erforderlich).

    ![Antennen](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x3000-antennas.jpg){class="gl-80-desktop glboxshadow"}

2. Richten Sie die Antennen für eine optimale Leistung nach oben aus.

## Externe Antennen wechseln

Sie können zusätzliche Wi-Fi- oder 5G-NR-Antennen kaufen, um die im Lieferumfang enthaltenen Antennen zu ersetzen. So können Sie Ihre Netzwerkeinstellungen individuell anpassen.

Es wird jedoch empfohlen, die Originalantennen zu verwenden, da diese vor dem Versand mehrfach getestet wurden, um eine optimale Kompatibilität sicherzustellen.

Wenn Sie lieber andere Antennen verwenden möchten, beachten Sie die folgenden Abschnitte, damit Sie Antennen mit den passenden Anschlüssen kaufen.

### Wi-Fi-Antennen

Die Basis der Wi-Fi-Antenne verwendet einen **RP-SMA-Buchsenanschluss** (Außengewinde und Innenstift). Erforderlich ist eine Wi-Fi-Antenne mit **RP-SMA-Stecker** (Innengewinde und Innenöffnung).

![RP-SMA-Anschlüsse](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/rp-sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### 5G-NR-Antennen

Die Basis der 5G-NR-Antenne verwendet einen **SMA-Buchsenanschluss** (Außengewinde und Innenöffnung). Erforderlich ist eine 5G-NR-Antenne mit **SMA-Stecker** (Innengewinde und Innenstift).

![SMA-Anschlüsse](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### Fehlerbehebung

Wenn Sie die falschen Antennen gekauft haben, können Sie einen Adapter in Betracht ziehen, der zwischen den Anschlüssen umsetzt (z. B. SMA-RP male auf SMA female).
