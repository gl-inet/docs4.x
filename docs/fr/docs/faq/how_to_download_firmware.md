# Comment télécharger le firmware de mon routeur GL.iNet ?

Si vous souhaitez mettre à jour votre routeur (ou le restaurer via U-Boot), téléchargez les fichiers du firmware en suivant les étapes ci-dessous :

1. Dans le [centre de téléchargement des firmwares GL.iNet](https://dl.gl-inet.com), utilisez la barre de recherche pour sélectionner le modèle de votre routeur.
2. Sélectionnez un onglet en haut :
    * **Stable** : version officielle pour une utilisation quotidienne, recommandée pour la plupart des utilisateurs.
    * **Beta** (disponible uniquement pour certains modèles de routeur) : version préliminaire encore en cours de développement.
    * **Snapshot** (disponible uniquement pour certains modèles de routeur) : compilation quotidienne incluant les dernières mises à jour.
3. Cliquez sur le bouton **Download**.
