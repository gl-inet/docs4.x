# UI Languages

Questa funzione e' stata aggiunta dal firmware v4.6.

---

Puoi fare clic sul pulsante della lingua nell'angolo in alto a destra per cambiare o aggiungere lingue dell'interfaccia.

Per aggiungere ulteriori lingue dell'interfaccia o aggiornare il testo UI, fai clic su **Add Languages**.

![languages list](https://static.gl-inet.com/docs/router/en/4/interface_guide/ui_languages/add_languages.jpg){class="glboxshadow"}

Nella finestra pop-up, seleziona la lingua desiderata e fai clic su **Update**. Il router scarichera' automaticamente l'ultimo language pack dal nostro repository. I language pack non installati verranno rimossi, tranne l'inglese.

Se vuoi ottenere automaticamente il language pack piu' recente per correggere potenziali problemi di testo, abilita **Enable Auto Update** e imposta l'orario di aggiornamento. Il router scarichera' automaticamente l'ultimo language pack all'orario specificato.

![languages list](https://static.gl-inet.com/docs/router/en/4/interface_guide/ui_languages/languages_list.png){class="glboxshadow"}
