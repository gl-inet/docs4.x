# Licencia DPI

DPI (Deep Packet Inspection) es una capacidad central de la gestión inteligente de redes. Puede superar la limitación de los routers tradicionales, que solo identifican direcciones de origen o destino, analizar en profundidad la carga útil de los paquetes de datos e identificar con precisión las aplicaciones y sitios web a los que acceden los usuarios mediante la comparación con una biblioteca de firmas, lo que permite una clasificación y un control del tráfico más precisos.

Integrado con [Netify](https://www.netify.ai/){target="\_blank"}, el DPI de GL.iNet adopta un complemento embebido ligero para un despliegue eficiente. Gracias a la base de datos de firmas de Netify, actualizada en línea, permite una gestión fiable y hace que el control de red sea más preciso y eficiente.

**Nota**:

1. Actualmente, las funciones DPI solo están disponibles en **GL-MT5000 (Brume 3)**.

2. Las funciones DPI no surtirán efecto cuando el router esté en modo Drop-in Gateway.

---

Inicie sesión en el panel de administración web del router y vaya a **FLOW CONTROL** > **DPI License**.

Inicie la **30-Day Trial** para disfrutar de las funciones DPI, incluidas **Data Statistics**, **Content Filter** y **QoS**.
