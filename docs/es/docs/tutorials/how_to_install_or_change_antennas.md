# Cómo instalar o cambiar las antenas externas de routers celulares

Este tutorial presenta la instalación y sustitución de antenas externas para algunos routers celulares.

## Instalar antenas externas

### Mudi 7

Mudi 7 (GL-E5800) está equipado con dos puertos de antena TS-9 que admiten frecuencias celulares de 600 MHz a 6 GHz. Puede conectar antenas externas (no incluidas) para mejorar la señal celular en zonas con poca cobertura. Es ideal para usarlo en vehículos recreativos, sótanos y entornos interiores para ampliar el alcance y garantizar una conexión estable.

![e5800 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/e5800-antenna.png){class="glboxshadow"}

Los puertos TS-9 del Mudi 7 son conectores hembra (orificio interno). Requieren antenas con conector macho TS-9 (pin interno).

![ts-9 connector](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/ts9_connector.png){class="glboxshadow" width="328"}

### Spitz AX / Puli AX

Spitz AX (GL-X3000) y Puli AX (GL-XE3000) incluyen las siguientes antenas externas:

* 2 antenas Wi-Fi (RP-SMA, 2.4G~2.5GHz, 5.15~5.84GHz)
* 4 antenas 5G NR (SMA, 600MHz~6GHz)

Siga estos pasos para instalar las antenas externas.

1. Conecte las antenas, sin importar el orden.

    * Conecte las dos antenas Wi-Fi 6 idénticas a los puertos **Wi-Fi 1** y **Wi-Fi 2**.
    * Conecte las cuatro antenas 5G NR idénticas a **MAIN**, **DIV**, **MIMO 1** y **MIMO2**.

    ![x3000 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x3000-antennas.jpg){class="gl-80-desktop glboxshadow"}

2. Coloque las antenas apuntando hacia arriba para obtener el mejor rendimiento.

### Spitz Plus

Spitz Plus (GL-X2000) incluye las siguientes antenas externas:

* 2 antenas Wi-Fi (RP-SMA, 2.4G~2.5GHz, 5.15~5.84GHz)
* 2 antenas 4G (SMA, 600MHz~3.8GHz)

Siga estos pasos para instalar las antenas externas.

1. Conecte las antenas, sin importar el orden.

    - Conecte las dos antenas Wi-Fi idénticas a los puertos **Wi-Fi 6**.
    - Conecte las dos antenas 4G idénticas al puerto **4G**.

    ![x2000 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x2000-back.png){class="gl-80-desktop glboxshadow"}

2. Coloque las antenas apuntando hacia arriba para obtener el mejor rendimiento.

### Spitz

Spitz (GL-X750/GL-X750V2) incluye 2 antenas externas 4G (SMA, 700M~2700MHz).

Siga estos pasos para instalar las antenas externas.

1. Conecte las dos antenas 4G idénticas a los puertos de antena.

    ![x750v2 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x750v2_back.png){class="gl-80-desktop glboxshadow"}

2. Coloque las antenas apuntando hacia arriba para obtener el mejor rendimiento.

### Collie

Collie (GL-X300B-RS485) incluye las siguientes antenas:

* 1 antena externa 4G
* 2 antenas Wi-Fi externas

Siga estos pasos para instalar las antenas.

1. Conecte las antenas, sin importar el orden.

    - Conecte las dos antenas Wi-Fi idénticas a los puertos **2.4G**.
    - Conecte la antena 4G al puerto **4G**.

    ![x300b antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x300b-rs485.png){class="gl-80-desktop glboxshadow"}

    ![x300b antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x300b-rs485-back.png){class="gl-80-desktop glboxshadow"}

2. Coloque las antenas apuntando hacia arriba para obtener el mejor rendimiento.

## Cambiar antenas externas

Las siguientes instrucciones se aplican a todos los routers celulares con puertos para antenas externas, excepto Mudi 7 (GL-E5800). Haga clic [aquí](#mudi-7){target="_blank"} para ver la instalación de antenas externas en Mudi 7.

Puede comprar antenas Wi-Fi o 4G/5G NR adicionales para sustituir las incluidas en el paquete, lo que le permite personalizar la configuración de la red.

No obstante, recomendamos usar las antenas originales, ya que se han sometido a amplias pruebas antes del envío para garantizar un rendimiento y una compatibilidad óptimos.

Si prefiere usar antenas de terceros, consulte las secciones siguientes para asegurarse de elegir antenas con el tipo de conector correcto.

**Consejo**: Si compró antenas incorrectas, puede considerar adquirir un adaptador que convierta entre conectores, por ejemplo, de SMA-RP macho a SMA hembra.

### Antenas Wi-Fi

La base de la antena Wi-Fi utiliza un **conector RP-SMA hembra** (rosca exterior y pin interior). Requiere una antena Wi-Fi con **conector RP-SMA macho** (rosca interior y orificio interior).

![rp-sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/rp-sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### Antenas 5G NR

La base de la antena 5G NR utiliza un **conector SMA hembra** (rosca exterior y orificio interior). Requiere una antena 5G NR con **conector SMA macho** (rosca interior y pin interior).

![sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/sma-connectors.jpg){class="gl-80-desktop glboxshadow"}
