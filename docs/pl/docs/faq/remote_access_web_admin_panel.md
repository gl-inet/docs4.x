# Jak uzyskać zdalny dostęp do panelu administracyjnego WWW routera GL.iNet?

- **Metoda 1**: Jeśli sieć Twojego routera ma [publiczny adres IP](../tutorials/how_to_check_if_isp_assigns_you_a_public_ip_address.md), skonfiguruj [WireGuard Server](../interface_guide/wireguard_server.md) i użyj WireGuard, aby uzyskać dostęp do panelu administracyjnego WWW routera. Jeśli nie masz publicznego adresu IP, wypróbuj pozostałe metody poniżej.
- **Metoda 2**: [Powiąż urządzenie z GoodCloud](../interface_guide/cloud.md#bind-devices-to-goodcloud), a następnie zaloguj się w [serwisie GoodCloud](https://www.goodcloud.xyz){target="_blank"}, aby [zdalnie uzyskać dostęp do panelu administracyjnego WWW routera](../interface_guide/cloud.md#remote-gui).
- **Metoda 3**: Jeśli chcesz skorzystać z rozwiązania SDWAN, polecamy [AstroWarp](https://www.astrowarp.net){target="_blank"}.
- **Metoda 4**: Jeśli chcesz skorzystać z rozwiązania reverse proxy, polecamy [AstroRelay](https://www.astrorelay.com){target="_blank"}.
