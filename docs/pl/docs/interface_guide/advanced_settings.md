# Zaawansowane ustawienia

Po lewej stronie panelu administracyjnego WWW przejdź do **SYSTEM** -> **Advanced Settings**.

Zaawansowane ustawienia możesz zmieniać za pomocą LuCI, domyślnego interfejsu WWW systemu OpenWrt.

![advanced settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/go_to_luci.png){class="glboxshadow"}

LuCI jest otwartym, niezależnym projektem rozwijanym przez OpenWrt i jest udostępniane w obecnej postaci. GL.iNet nie odpowiada za utrzymanie LuCI.

Kliknij przycisk **Go To LuCI**, a zostaniesz przekierowany na stronę logowania LuCI.

![luci login](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/luci_login.jpg){class="glboxshadow"}

Wprowadź hasło logowania. Jest ono takie samo jak hasło do panelu administracyjnego WWW.

Po zalogowaniu zostanie wyświetlona strona LuCI. Poniżej pokazano przykład strony LuCI dla GL-AX1800 (Flint).

![luci logged in](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/luci_logged_in.jpg){class="glboxshadow"}
