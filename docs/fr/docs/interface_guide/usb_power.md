# USB & Power

**Remarque** : cette page n'est disponible que sur **GL-E5800 (Mudi 7)**.

Dans la partie gauche du panneau d'administration web, accédez à **SYSTEM** -> **USB & Power**.

Cette page vous permet de configurer les paramètres liés à l'USB et à l'alimentation du routeur, comme le protocole USB, le sens d'alimentation, les délais d'inactivité et le comportement en veille.

## Paramètres USB

Cette section vous permet de personnaliser le comportement des ports USB du routeur.

![usb settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_usb.png){class="glboxshadow"}

- **USB Protocol Switch** : permet de basculer le port USB entre les protocoles USB 2.0 et USB 3.1.

- **Dual Role USB Mode** : permet de sélectionner le mode de fonctionnement USB dans la liste déroulante. Vous pouvez choisir **Device** ou **Host**.

    ![usb role](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/usb_role.png){class="glboxshadow"}

- **Power Direction** : permet de choisir la priorité d'alimentation du port USB dans la liste déroulante. Vous pouvez sélectionner **Input Priority** ou **Output Priority**.

- **Power Threshold** : permet de définir un seuil de puissance spécifique pour le port USB afin de contrôler le niveau minimal requis pour les opérations USB, ce qui aide à préserver l'autonomie de la batterie.

## Paramètres d'alimentation

Cette section vous aide à optimiser la consommation d'énergie et le comportement de l'appareil.

![power settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_power.png){class="glboxshadow"}

- **Wi-Fi Idle Timeout** : définit la durée d'inactivité (de 10 minutes à 2 heures, ou jamais) après laquelle le Wi-Fi passe en veille.

- **Ethernet Idle Timeout** : définit la durée d'inactivité (de 10 minutes à 2 heures, ou jamais) au terme de laquelle Ethernet passe en veille.

- **Power Off Timeout** : définit le délai (de 1 heure à 12 heures, ou jamais) avant l'arrêt automatique du routeur lorsqu'il est inactif.

- **Power On with Charger** : active ou désactive l'allumage automatique du routeur lorsqu'il est branché à un chargeur.

- **Auto Standby** : active ou désactive la mise en veille automatique afin d'économiser l'énergie.
