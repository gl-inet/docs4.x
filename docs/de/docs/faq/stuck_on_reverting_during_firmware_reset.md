# Hängt bei „Reverting“ während des Firmware-Zurücksetzens

Wenn Ihr Router während eines Firmware-Zurücksetzens bei „Reverting“ hängen bleibt, befolgen Sie die folgenden Schritte, um das Problem zu beheben.

![stuck on reverting](https://static.gl-inet.com/docs/router/en/4/faq/troubleshooting/stuck_on_reverting/reverting.png){class="glboxshadow"}

Prüfen Sie nach jedem Schritt, ob das Problem behoben ist. Wenn ja, überspringen Sie die verbleibenden Schritte.

1. Warten Sie mindestens 10 Minuten, um zu prüfen, ob der Zurücksetzungsvorgang von selbst abgeschlossen wird. Dieser Vorgang kann manchmal länger dauern als erwartet.

2. Wenn der Vorgang nach 10 Minuten nicht abgeschlossen ist, schalten Sie Ihren Router aus, indem Sie das Stromkabel abziehen. Warten Sie 30 Sekunden, schließen Sie das Stromkabel wieder an und warten Sie, bis der Router hochgefahren ist.

3. Öffnen Sie einen Webbrowser, rufen Sie `192.168.8.1` auf und melden Sie sich an. Wenn das Zurücksetzen erfolgreich war, werden Sie zum Einrichtungsassistenten weitergeleitet.

4. Aktivieren Sie den privaten/Inkognito-Modus Ihres Browsers und rufen Sie `192.168.8.1` auf. Wenn das Zurücksetzen erfolgreich war, werden Sie zum Einrichtungsassistenten weitergeleitet.

5. Verwenden Sie einen anderen Browser (z. B. Chrome, Edge, Safari), um `192.168.8.1` aufzurufen. Wenn das Zurücksetzen erfolgreich war, werden Sie zum Einrichtungsassistenten weitergeleitet.

6. Wenn Sie weiterhin nicht auf das Web-Admin-Panel des Routers zugreifen können oder der Router nicht ordnungsgemäß funktioniert, versuchen Sie, den Router mithilfe [dieser Anleitung](repair_network_or_reset_firmware.md/#reset-to-factory) auf die Werkseinstellungen zurückzusetzen.
