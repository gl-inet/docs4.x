# Plug-ins

Dans le panneau d'administration web, accédez à **APPLICATIONS** -> **Plug-ins**.

Cette page vous permet de gérer les paquets OpenWrt. Vous pouvez installer ou supprimer des paquets selon vos besoins.

Il est recommandé de cliquer sur le bouton **Update** avant d'utiliser des plug-ins tiers. Cependant, **ne mettez pas à jour les plug-ins préinstallés du firmware officiel GL.iNet**, tels que `nginx`, `adguard home`, `tailscale`, etc.

![Plug-ins](https://static.gl-inet.com/docs/router/en/4/tutorials/plug-ins/plug-ins.png){class="glboxshadow"}

Les plug-ins tiers installés manuellement par les utilisateurs peuvent être mis à jour ; toutefois, GL.iNet ne saurait être tenu responsable d'éventuels problèmes de sécurité liés à des plug-ins tiers. Pour toute question, veuillez contacter les auteurs du plug-in tiers concerné.
