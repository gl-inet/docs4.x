# Bloqué sur "Installing" pendant la mise à jour du firmware

Si votre routeur reste bloqué sur "Installing" pendant une mise à jour du firmware, suivez les étapes ci-dessous pour résoudre le problème.

![stuck on installing](https://static.gl-inet.com/docs/router/en/4/faq/troubleshooting/stuck_on_installing/installing.png){class="glboxshadow"}

Après chaque étape, vérifiez si le problème est résolu. Si c'est le cas, ignorez les étapes restantes.

1. Vérifiez que la mise à jour du firmware a réussi.

    Ouvrez d'abord un navigateur Web, accédez à `192.168.8.1` et connectez-vous.
    
    Vérifiez ensuite la version du firmware en haut à gauche. Si elle correspond à la version cible, la mise à jour a réussi. Sinon, passez à l'étape suivante.

2. Assurez-vous que la connexion réseau est stable, puis téléversez de nouveau le firmware.

3. Utilisez un autre navigateur (par exemple, Edge ou Safari) pour téléverser de nouveau le firmware.
