# Comment connecter au Wi-Fi un appareil ne disposant que d'un port Ethernet avec un routeur GL.iNet ?

Si votre appareil prend uniquement en charge une connexion Ethernet mais doit se connecter à un réseau sans fil, un routeur GL.iNet peut combler ce besoin.

Un routeur GL.iNet peut servir de pont Ethernet-vers-Wi-Fi pour les appareils qui ne disposent que d'une connexion filaire, leur permettant ainsi de rejoindre un réseau sans fil. Pour cela, il suffit de configurer le routeur en mode Repeater.

Veuillez connecter votre appareil au port LAN du routeur, puis configurer [Repeater](../interface_guide/internet_repeater.md).
