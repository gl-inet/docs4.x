# Czy podczas korzystania z VPN trzeba konfigurować Ethernet WAN?

Nie. Gdy skonfigurujesz router GL.iNet jako klient VPN, możesz użyć dowolnego źródła WAN. Ruch z sieci LAN i WLAN będzie chroniony przez tunel VPN.

## Konfiguracja przez repeater Wi-Fi

Możesz połączyć router GL.iNet (skonfigurowany jako klient VPN) z głównym routerem bezprzewodowo. W takiej konfiguracji router GL.iNet będzie działał jako repeater Wi-Fi.

![repater](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/repeater.jpg){class="glboxshadow"}

## Konfiguracja przez kabel Ethernet

Możesz połączyć router GL.iNet (skonfigurowany jako klient VPN) z głównym routerem za pomocą kabla Ethernet.

![ethernet](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/ethernet.jpg){class="glboxshadow"}

## Konfiguracja przez USB Tethering

Możesz połączyć router GL.iNet (skonfigurowany jako klient VPN) z telefonem za pomocą kabla USB. To rozwiązanie jest również nazywane USB Tethering.

![tethering](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/tethering.jpg){class="glboxshadow"}
