# UI-Sprachen

Diese Funktion wurde ab Firmware v4.6 hinzugefügt.

---

Sie können oben rechts auf die Sprachschaltfläche klicken, um UI-Sprachen zu wechseln oder hinzuzufügen.

Um zusätzliche UI-Sprachen hinzuzufügen oder UI-Texte zu aktualisieren, klicken Sie auf **Add Languages**.

![languages list](https://static.gl-inet.com/docs/router/en/4/interface_guide/ui_languages/add_languages.jpg){class="glboxshadow"}

Wählen Sie im Pop-up-Fenster die gewünschte Sprache aus und klicken Sie auf **Update**. Der Router lädt dann automatisch das neueste Sprachpaket aus unserem Repository herunter. Nicht installierte Sprachpakete werden entfernt (außer Englisch).

Wenn Sie automatisch das neueste Sprachpaket beziehen möchten, um mögliche Textprobleme zu beheben, aktivieren Sie **Enable Auto Update** und legen Sie die Aktualisierungszeit fest. Der Router lädt dann zum angegebenen Zeitpunkt automatisch das neueste Sprachpaket herunter.

![languages list](https://static.gl-inet.com/docs/router/en/4/interface_guide/ui_languages/languages_list.png){class="glboxshadow"}
