# AdGuard Home

AdGuard Homeは、広告とトラッキングをブロックするネットワーク全体用ソフトウェアです。設定が完了すると、追加のクライアントサイドソフトウェアを必要とせず、家庭内ネットワーク上のすべてのデバイスを保護します。

## 設定

ルーターのWeb管理パネルにログインし、**APPLICATIONS** -> **AdGuard Home** に移動します。

スイッチをオンにしてAdGuard Homeを有効にし、**Apply** をクリックします。

![adguard home apply](https://static.gl-inet.com/docs/router/en/4/interface_guide/adguard_home/apply.png){class="glboxshadow"}

- **AdGuard Home Handle Client Requests**: 有効にすると、クライアントデバイスからのDNSクエリは AdGuard Home によって直接処理されます。この場合、ドメインベースの VPN ポリシーとペアレンタルコントロールのルールは機能しなくなります。

有効化すると、このページには AdGuard Home の API を通じて DNS クエリ数やブロックされたドメイン数などの統計情報が表示されます。

![adguard home web panel](https://static.gl-inet.com/docs/router/en/4/interface_guide/adguard_home/adguardhome_web_panel.png){class="glboxshadow"}

ページ上部の **Settings Page** をクリックします。

![adguard home started](https://static.gl-inet.com/docs/router/en/4/interface_guide/adguard_home/settings_page.png){class="glboxshadow"}

AdGuard Home の公式設定ページへリダイレクトされ、フィルタルールなどの高度な設定を行えます。

![adguard home settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/adguard_home/adguardhome_dashboard.png){class="glboxshadow"}

ご不明な点がある場合は、[AdGuard Home Support Center](https://adguard.com/en/support.html){target="_blank"} をご利用ください。
