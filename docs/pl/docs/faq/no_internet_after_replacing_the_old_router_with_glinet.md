# Co zrobić, jeśli po wymianie starego routera na router GL.iNet nie ma dostępu do Internetu

Jeśli po wymianie starego routera na router GL.iNet nie możesz uzyskać dostępu do Internetu, nowy router może nie otrzymywać adresu IP (przez DHCP lub PPPoE), gdy jego port WAN jest podłączony do modemu.

Twój dostawca usług internetowych może mieć włączone **powiązanie adresu MAC (MAC binding)**, przez co z siecią może łączyć się tylko adres MAC poprzedniego routera.

Wypróbuj poniższe metody, aby rozwiązać ten problem.

- Sklonuj adres MAC starego routera do nowego routera GL.iNet.
- Uruchom ponownie modem. Jeśli problem nadal występuje, wyłącz go na 30 minut, a następnie włącz ponownie.
- Skontaktuj się z dostawcą usług internetowych i poproś o zresetowanie modemu.
