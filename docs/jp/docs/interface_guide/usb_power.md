# USB & Power

**Note**: このページは **GL-E5800 (Mudi 7)** でのみ利用できます。

Web管理パネルの左側で、**SYSTEM** -> **USB & Power** に移動します。

このページでは、USB プロトコル、給電方向、アイドルタイムアウト、スタンバイ動作など、ルーターの USB 関連設定と電源管理を構成できます。

## USB設定

このセクションでは、ルーターの USB ポートの動作をカスタマイズできます。

![usb settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_usb.png){class="glboxshadow"}

- **USB Protocol Switch**: USB ポートで USB 2.0 と USB 3.1 protocol を切り替えます。

- **Dual Role USB Mode**: ドロップダウンメニューから USB の動作モードを選択します。**Device** または **Host** に設定できます。

    ![usb role](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/usb_role.png){class="glboxshadow"}

- **Power Direction**: ドロップダウンメニューから USB ポートの電源優先順位を選択します。**Input Priority** または **Output Priority** に設定できます。

- **Power Threshold**: USB ポートのしきい値を割合で設定し、USB 動作に必要な最小電力レベルを制御して、バッテリー駆動時間の維持に役立てます。

## 電源設定

このセクションでは、消費電力とデバイス動作を最適化できます。

![power settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_power.png){class="glboxshadow"}

- **Wi-Fi Idle Timeout**: Wi-Fi がスタンバイに入るまでのアイドル時間を設定します（10 分〜2 時間、または never）。

- **Ethernet Idle Timeout**: Ethernet がスタンバイに切り替わるまでのアイドル時間を設定します（10 分〜2 時間、または never）。

- **Power Off Timeout**: アイドル時にルーターが自動的に電源オフになるまでの遅延時間を設定します（1 時間〜12 時間、または never）。

- **Power On with Charger**: このスイッチで、充電器に接続したときにルーターを自動的に起動するかどうかを切り替えます。

- **Auto Standby**: このスイッチで、省電力のための自動スタンバイを有効または無効にします。
