# Display Management

**Nota**: Esta función solo está disponible para **GL-BE3600 (Slate 7)** y **GL-E5800 (Mudi 7)**.

En el lado izquierdo del panel de administración web, vaya a **SYSTEM** -> **Display Management**.

Esta página le permite gestionar la pantalla táctil y sus ajustes relacionados.

## GL-BE3600

![display management](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_display.png){class="glboxshadow"}

- **Function Management**: gestione qué funciones se muestran en la pantalla táctil.

  ![function](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_function.png){class="glboxshadow"}

- **Lock Screen**: personalice los ajustes de la pantalla de bloqueo, incluida la pantalla de activación y el fondo de pantalla.

  ![lock screen](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_lockscreen.png){class="glboxshadow"}

  Cambie `wallper` a **Customize** y podrá cargar una imagen de tamaño no superior a 5 MB.

  ![wallpaper customize](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_wallpaper_custom.png){class="glboxshadow"}

- **Brightness**: ajuste el brillo de la pantalla táctil. Use el control deslizante o introduzca un nivel específico, de 1 a 10, para adaptarse a distintas condiciones de iluminación.
- **Auto Lock**: establezca el tiempo de espera para que la pantalla se bloquee automáticamente cuando no haya actividad. El rango va de 1 a 30 minutos.
- **Screen Always On**: active o desactive esta opción para decidir si la pantalla táctil permanece encendida de forma continua o se apaga tras un periodo de inactividad.
- **Enable Screen Passcode**: añada una capa extra de seguridad habilitando un código de acceso para la pantalla táctil, solo números de 4 dígitos.

  ![screen passcode](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/be3600_passcode.png){class="glboxshadow"}

## GL-E5800

![display management](https://static.gl-inet.com/docs/router/en/4/interface_guide/display_management/e5800_display.png){class="glboxshadow"}

- **Wallpaper**: personalice el fondo de pantalla y el estilo de activación de la pantalla.
- **Brightness**: ajuste el brillo de la pantalla táctil. Use el control deslizante o introduzca un porcentaje para adaptarlo a distintas condiciones de iluminación.
- **Personalised Signature**: añada un texto personalizado a la pantalla táctil para mostrar su estilo único o para una identificación rápida.
- **Auto Lock**: establezca el tiempo de espera para que la pantalla se bloquee automáticamente cuando no haya actividad. El rango va de 15 segundos a 5 minutos.
- **Passcode**: establezca un código de acceso para la pantalla táctil como capa adicional de seguridad.
