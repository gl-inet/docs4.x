# Devo configurare Ethernet WAN quando uso la VPN?

La risposta è no. Quando imposti il router GL.iNet come client VPN, puoi configurare qualsiasi sorgente WAN. La tua LAN e la tua WLAN saranno coperte dal tunnel VPN.

## Configurazione tramite Wi-Fi Repeater

Puoi collegare il router GL.iNet (impostato come client VPN) al router principale in modalità wireless; in questo modo il router GL.iNet funzionerà come Wi-Fi Repeater.

![repater](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/repeater.jpg){class="glboxshadow"}

## Configurazione tramite cavo Ethernet

Puoi collegare il router GL.iNet (impostato come client VPN) al router principale tramite un cavo Ethernet.

![ethernet](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/ethernet.jpg){class="glboxshadow"}

## Configurazione tramite USB Tethering

Puoi collegare il router GL.iNet (impostato come client VPN) al telefono tramite un cavo USB; questa modalità è nota anche come USB Tethering.

![tethering](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/tethering.jpg){class="glboxshadow"}
