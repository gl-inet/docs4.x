# AdGuard Home

AdGuard Homeは、広告とトラッキング（追跡）をブロックするネットワーク全体用ソフトウェアです。設定が完たすると、追加のクライアントサイドソフトウェアを必要とせず、家庭内ネットワーク上のすべてのデバイスを保護します。

## 設定

ルーターのWeb管理パネルにログイン -> アプリケーション -> AdGuard Home。スイッチをオンにしてAdGuard Homeを有効にし、適用をクリックします。

![adguard home apply](https://static.gl-inet.com/docs/router/en/4/interface_guide/adguard_home/apply.png){class="glboxshadow"}

- AdGuard Homeがクライアントリクエストを処理: このオプションを有効にすると、クライアントデバイスからのDNSクエリはAdGuard Homeによって直接処理されます。AdGuard HomeはクライアントによるDNSクエリログを表示しますが、ドメインに基づくVPNポリシーが機できるしなくなります。

このページには、DNSクエリ、ブロックされたドメインなどの統計情報が表示されます（AdGuard HomeのAPI経より）。

![adguard home web panel](https://static.gl-inet.com/docs/router/en/4/interface_guide/adguard_home/adguardhome_web_panel.png){class="glboxshadow"}

次に、**設定ページ** をクリックしてください。

![adguard home started](https://static.gl-inet.com/docs/router/en/4/interface_guide/adguard_home/settings_page.png){class="glboxshadow"}

AdGuard Homeの公式設定ページにリダイレクトされます。ここでは、フィルタルールなどの詳細設定を行うことができます。

![adguard home settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/adguard_home/adguardhome_dashboard.png){class="glboxshadow"}

ご不明な時がございましたら、[Adguard Homeサポートセンター](https://adguard.com/en/support.html){target="_blank"}までお問い合わせください。
