# Jak pobrać firmware do routera GL.iNet?

Jeśli chcesz zaktualizować router (lub odzyskać go przez U-Boot), pobierz pliki firmware, wykonując poniższe kroki:

1. W [centrum pobierania firmware GL.iNet](https://dl.gl-inet.com) kliknij pole wyszukiwania i wybierz model swojego routera.
2. Wybierz kartę u góry:
    * **Stable:** oficjalne wydanie do codziennego użytku, zalecane dla większości użytkowników.
    * **Beta** (dostępne tylko dla wybranych modeli routerów): wersja przedpremierowa, która nadal jest rozwijana.
    * **Snapshot** (dostępne tylko dla wybranych modeli routerów): codzienna kompilacja zawierająca najnowsze aktualizacje.
3. Kliknij przycisk **Download**.
