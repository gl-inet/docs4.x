# Come collegare a una rete Wi-Fi un dispositivo solo Ethernet con un router GL.iNet?

Se il tuo dispositivo supporta solo la connettività Ethernet ma deve collegarsi a una rete wireless, un router GL.iNet può colmare questa esigenza.

Un router GL.iNet può funzionare come bridge Ethernet-Wi-Fi per dispositivi che dispongono solo di una connessione cablata, permettendo loro di collegarsi a una rete wireless. Questo si ottiene configurando il router come Repeater.

Collega il dispositivo alla porta LAN del router e poi configura il [repeater](../interface_guide/internet_repeater.md).
