# Bloqueado en "Installing" durante la actualización del firmware

Si su router se queda bloqueado en "Installing" durante una actualización de firmware, siga los pasos que se indican a continuación para resolver el problema.

![stuck on installing](https://static.gl-inet.com/docs/router/en/4/faq/troubleshooting/stuck_on_installing/installing.png){class="glboxshadow"}

Después de cada paso, compruebe si el problema se ha resuelto. Si es así, omita los pasos restantes.

1. Verifique que la actualización del firmware se haya completado correctamente.

   Primero, abra un navegador web, acceda a `192.168.8.1` e inicie sesión.

   Luego, compruebe la versión del firmware en la esquina superior izquierda. Si coincide con la versión de destino, la actualización se realizó correctamente. Si no es así, continúe con el paso siguiente.

2. Asegúrese de tener una conexión de red estable y vuelva a cargar el firmware.

3. Use un navegador diferente, por ejemplo, Edge o Safari, para volver a cargar el firmware.
