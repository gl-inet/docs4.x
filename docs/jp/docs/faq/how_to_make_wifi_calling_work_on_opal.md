# Opal (GL-SFT1200) でWi-Fi通話を動作させる方法

## ファームウェアがV4.3.17で降であることを確認する

Web管理パネルで、システム -> アップグレードに移動し、ファームウェアを最も新バージョン（V4.3.17で降）にアップグレードします。ファームウェアのダウンロードリンクは[こちら](https://dl.gl-inet.com/)です（オンラインアップグレードまたはローカルアップグレードの两者できます）

![opal4317](https://static.gl-inet.com/docs/router/en/4/faq/wifi_call/opal4317.jpg){class="glboxshadow}

## ネットワークアクセラレーションを無効にする

ファームウェアアップグレード後、Web管理パネルで、ネットワーク -> ネットワークアクセラレーションに移動し、無効にします。

![networkacc](https://static.gl-inet.com/docs/router/en/4/faq/wifi_call/netacc.jpg){class="glboxshadow}

私たちの経験とお客様からフィードバックによると、ネットワーク/ハードウェアアクセラレーションを無効にすると、Wi-Fi通話が正常に動作するようになります。

この解決策がデバイスで機能しない場合は、[コミュニティフォーラム](https://forum.gl-inet.com){target="_blank"}または[お問い合わせ](https://www.gl-inet.com/contacts/){target="_blank"}をご覧ください。
