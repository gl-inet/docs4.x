# ¿Cómo descargar el firmware para mi router GL.iNet?

Si desea actualizar su router, o recuperarlo mediante U-Boot, descargue los archivos de firmware siguiendo estos pasos:

1. En el [centro de descarga de firmware de GL.iNet](https://dl.gl-inet.com), seleccione la barra de búsqueda y su modelo de router.
2. Seleccione una pestaña en la parte superior:
   - **Stable:** La versión oficial para uso diario y recomendada para la mayoría de los usuarios.
   - **Beta**, disponible solo para algunos modelos de router: La versión preliminar que todavía está en desarrollo.
   - **Snapshot**, disponible solo para algunos modelos de router: La compilación diaria que incluye las últimas actualizaciones.
3. Haga clic en el botón **Download**.
