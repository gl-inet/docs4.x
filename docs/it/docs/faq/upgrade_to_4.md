# Come aggiornare al firmware 4.x?

**Questo documento è obsoleto e non viene più mantenuto. Fai clic [qui](../interface_guide/upgrade.md) per la guida di aggiornamento più recente.**

---

Segui i passaggi riportati di seguito per aggiornare il router dal firmware 3.x al 4.x.

Nota: non mantenere le impostazioni durante l'aggiornamento da v3.x a v4.x, perché questo può causare instabilità. Esegui il backup delle impostazioni critiche prima dell'aggiornamento.

GL-B1300 e GL-S1300 non supportano la funzione mesh nel firmware v4.x.

---

## Metodo 1. Aggiornamento locale

Puoi aggiornare il firmware tramite il pannello di amministrazione web.

1. Aggiorna il firmware all'ultima versione stabile 3.x.

2. Scarica il firmware 4.x [qui](https://dl.gl-inet.com){target="_blank"}. Assicurati di **scaricare quello per common upgrade**.

3. Accedi al pannello di amministrazione web, vai su **Upgrade** -> **Local Upgrade**. Carica il file firmware appena scaricato.

    **Nota:** il firmware 4.x non è compatibile con il firmware 3.x. Quando esegui l'aggiornamento dal firmware 3.x, **NON** mantenere le impostazioni.

    ![local upgrade](https://static.gl-inet.com/docs/router/en/4/tutorials/gl-ax1800_upgrade_to_4/ax1800_upgrade_4.png){class="glboxshadow gl-90-desktop"}

## Metodo 2. Aggiornamento U-Boot

1. Scarica il firmware 4.x [qui](https://dl.gl-inet.com){target="_blank"}. Assicurati di **scaricare quello per U-Boot**.

2. Fai riferimento a questa guida [U-Boot](debrick.md) per eseguire il flash del firmware.
