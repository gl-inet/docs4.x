# ¿Cómo acceder por SSH de forma remota al terminal del router GL.iNet?

- **Método 1**: Si la red de su router tiene una [IP pública](../tutorials/how_to_check_if_isp_assigns_you_a_public_ip_address.md), configure un [WireGuard Server](../interface_guide/wireguard_server.md) y use WireGuard para acceder por SSH al router. Si no tiene una IP pública, pruebe los otros métodos indicados a continuación.
- **Método 2**: [Vincule su dispositivo a GoodCloud](../interface_guide/cloud.md#setup), luego podrá [acceder remotamente por SSH al router](../interface_guide/cloud.md#remote-access-routers-terminal) iniciando sesión en el [sitio web de GoodCloud](https://www.goodcloud.xyz){target="\_blank"}.
- **Método 3**: Si desea usar una solución SDWAN, le sugerimos [AstroWarp](https://www.astrowarp.net){target="\_blank"}.
- **Método 4**: Si desea usar una solución de proxy inverso, le sugerimos [AstroRelay](https://www.astrorelay.com){target="\_blank"}.
