# Come scaricare il firmware per il mio router GL.iNet?

Se vuoi aggiornare il router (o sbloccarlo tramite U-Boot), scarica i file del firmware seguendo questi passaggi:

1. Nel [centro download firmware GL.iNet](https://dl.gl-inet.com), seleziona la barra di ricerca e il modello del router.
2. Seleziona una scheda nella parte superiore:
    * **Stable:** la versione ufficiale per l'uso quotidiano e consigliata per la maggior parte degli utenti.
    * **Beta** (disponibile solo per alcuni modelli di router): la versione pre-release ancora in fase di sviluppo.
    * **Snapshot** (disponibile solo per alcuni modelli di router): la build giornaliera che include gli aggiornamenti più recenti.
3. Fai clic sul pulsante **Download**.
