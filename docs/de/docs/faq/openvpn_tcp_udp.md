# OpenVPN, TCP vs. UDP

Der OpenVPN-Server des GL.iNet-Routers unterstützt sowohl TCP- als auch UDP-Protokolle. Worin liegt der Unterschied?

TCP ist zuverlässiger, aber langsamer. UDP ist schneller, aber weniger zuverlässig.

Wenn Sie für eine Anwendung eine geringe Latenz und eine kontinuierliche Datenübertragung benötigen, ist UDP die bessere Wahl. Andernfalls ist TCP ein zuverlässiges Protokoll für die verlustfreie Datenübertragung.

UDP eignet sich besser für Gaming, Streaming oder die Nutzung von VoIP-Diensten. TCP eignet sich besser für E-Mails, Webbrowsing und Dateiübertragungen.

Wir empfehlen, zunächst das UDP-Protokoll auszuprobieren und nur dann zu TCP zu wechseln, wenn Verbindungsinstabilität, Paketverlust oder andere Zuverlässigkeitsprobleme auftreten.
