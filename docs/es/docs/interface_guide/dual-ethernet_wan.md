# WAN Ethernet dual

La función Dual-Ethernet WAN permite cambiar el puerto LAN1 predeterminado para usarlo como un segundo puerto WAN y así disponer de acceso dual por Ethernet. Esto proporciona una conectividad de respaldo fiable y, cuando es compatible, admite agregación de ancho de banda para entornos con alta demanda. También le permite conectarse simultáneamente a dos redes diferentes, por ejemplo una de trabajo y otra personal, lo que aporta mayor flexibilidad sin necesidad de hardware adicional.

## Modelos compatibles

??? "Modelos compatibles" - ※GL-E5800 (Mudi 7) - GL-MT3600BE (Beryl 7) - GL-MT5000 (Brume 3) - GL-BE6500 (Flint 3e) - GL-BE9300 (Flint 3) - GL-BE3600 (Slate 7) - GL-X2000 (Spitz Plus) - GL-MT6000 (Flint2) - GL-X3000 (Spitz AX) - GL-XE3000 (Puli AX)

    **Nota**: El GL-E5800 (Mudi 7) está equipado con un puerto Ethernet, LAN por defecto que puede cambiarse a WAN, y con un **puerto USB-C con OTG habilitado**. Para añadir un segundo puerto Ethernet para Dual-Ethernet WAN, conecte al puerto USB-C un adaptador USB‑C a Ethernet que se vende por separado.

??? "Modelos no compatibles" - GL-B3000 (Marble) - GL-AX1800 (Flint) - GL-MT2500/GL-MT2500A (Brume 2) - GL-MT3000 (Beryl AX) - GL-AXT1800 (Slate AX) - GL-SFT1200 (Opal) - GL-A1300 (Slate Plus) - GL-MT1300 (Beryl) - GL-E750/E750V2 (Mudi) - GL-AR750S (Slate) - GL-XE300 (Puli) - GL-X750 (Spitz) - GL-MT300N-V2 (Mango) - GL-AR300M Series (Shadow) - GL-B1300 (Convexa-B) - GL-X300B (Collie)

## Configuración

1. Inicie sesión en el panel de administración web del router.

2. Vaya a **NETWORK** -> **Port Management** -> pestaña **LAN**, cambie la propiedad del puerto a WAN y haga clic en **Apply**.

   ![dual ethernet wan 1](https://static.gl-inet.com/docs/router/en/4/tutorials/dual-ethernet_wan/dual_ethernet_1.png){class="glboxshadow"}

3. En la ventana emergente, haga clic en **Apply**.

   ![dual ethernet wan 2](https://static.gl-inet.com/docs/router/en/4/tutorials/dual-ethernet_wan/dual_ethernet_2.png){class="glboxshadow"}

4. Después de habilitar Dual-Ethernet WAN, puede configurar las funciones multi-wan [aquí](../interface_guide/multi-wan.md).
