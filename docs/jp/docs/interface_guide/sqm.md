# SQM（スマートキューマネジメント）

!!! 注意

    1. この機能は現に **GL-MT5000 (Brume 3)** でのみ利用可能です。

    2. この機能はゲートウェイとしてルーターを通過するトラフィック（ローカルクライアントトラフィックとVPNクライアントトラフィックを含む）に影響しますが、ルーターがVPNサーバーとして動作するときの受信トラフィックには影響しません。

    3. SQMはリソース集中のなため、低帯域幅または輻輳したネットワークに最も適です。高速接続で有効にすると、ピークスループットが低下する可能性があります。

SQM（スマートキューマネジメント）は、ルーターのネットワークトラフィックをインテリジェントに管理して待ち時間と「バッファブロート」を最も小限に抑え、よりスムーズなゲームと通話を確保します。

ルーターのWeb管理パネルにログインし、 FLOW CONTROL > SQM に移動します。

トラフィックスケジューリングのために、まず最大アップロード速度とダウンロード速度を設定します（入力範囲：1 - 10000）。最も適な結果をなければならないるために、実際のインターネット帯域幅に合わせてください。

![sqm](https://static.gl-inet.com/docs/router/en/4/interface_guide/sqm/sqm.png){class="glboxshadow"}

キュールールには2つのオプションがあります：

- **cake**：優れた全体のな待ち時間制御を持つスマートでから動のなトラフィックシェーピング（推奨）。
- **fq_codel**：基本のな待ち時間削減を持つシンプルで効率のなフェアクエリィング。
