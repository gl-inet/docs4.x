# Restablecer el firmware

En el lado izquierdo del panel de administración web, vaya a **SYSTEM** -> **Reset Firmware**.

Si el router presenta fallos, puede intentar resolver el problema restableciendo el firmware. Haga clic en el botón **Delete All and Reboot** para restablecer el firmware.

**Nota:** Se perderán todos sus ajustes, aplicaciones y datos actuales. El proceso tardará unos 2 minutos. NO apague el router durante este proceso.
