# Plug-ins

Öffnen Sie auf der linken Seite des Web-Admin-Panels **APPLICATIONS** -> **Plug-ins**.

Auf dieser Seite können Sie OpenWrt-Pakete verwalten. Sie können Pakete nach Bedarf installieren oder entfernen.

Es wird empfohlen, vor der Verwendung von Plug-ins von Drittanbietern auf **Update** zu klicken. **Aktualisieren Sie jedoch keine vorinstallierten Plug-ins aus der offiziellen GL.iNet-Firmware**, z. B. `nginx`, `adguard home`, `tailscale` usw.

![Plug-ins](https://static.gl-inet.com/docs/router/en/4/tutorials/plug-ins/plug-ins.png){class="glboxshadow"}

Von Benutzern manuell installierte Plug-ins von Drittanbietern können aktualisiert werden. GL.iNet übernimmt jedoch keine Haftung für Sicherheitsprobleme, die durch Plug-ins von Drittanbietern entstehen. Bei Fragen wenden Sie sich bitte an die jeweiligen Autoren der Drittanbieter-Plug-ins.
