# USB i zasilanie

**Uwaga**: Ta strona jest dostępna wyłącznie dla modelu **GL-E5800 (Mudi 7)**.

W lewym panelu bocznym interfejsu administracyjnego przejdź do **SYSTEM** -> **USB & Power**.

Ta strona umożliwia konfigurację ustawień USB i zarządzania zasilaniem routera, takich jak protokół USB, kierunek zasilania, limity czasu bezczynności oraz tryb gotowości.

## Ustawienia USB

Ta sekcja umożliwia dostosowanie działania portów USB routera.

![usb settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_usb.png){class="glboxshadow"}

- **USB Protocol Switch**: Przełączanie między protokołem USB 2.0 a USB 3.1 dla portu USB.

- **Dual Role USB Mode**: Wybierz tryb pracy USB z menu rozwijanego. Możesz ustawić tryb Device (urządzenie) lub Host.

    ![usb role](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/usb_role.png){class="glboxshadow"}

- **Power Direction**: Wybierz priorytet zasilania portu USB z menu rozwijanego. Możesz ustawić Input Priority (priorytet wejścia) lub Output Priority (priorytet wyjścia).

- **Power Threshold**: Ustaw próg zasilania USB (w procentach), który określa minimalny poziom energii wymagany do działania USB, co pomaga oszczędzać baterię.

## Ustawienia zasilania

Ta sekcja umożliwia optymalizację zużycia energii i działania urządzenia.

![power settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_power.png){class="glboxshadow"}

- **Wi-Fi Idle Timeout**: Ustaw czas bezczynności (od 10 minut do 2 godzin lub nigdy), po którym Wi-Fi przejdzie w tryb gotowości.

- **Ethernet Idle Timeout**: Ustaw czas bezczynności (od 10 minut do 2 godzin lub nigdy), po którym Ethernet przejdzie w tryb gotowości.

- **Power Off Timeout**: Ustaw opóźnienie (od 1 do 12 godzin lub nigdy), po którym router automatycznie wyłączy się w stanie bezczynności.

- **Power On with Charger**: Włącz lub wyłącz automatyczne uruchamianie routera po podłączeniu ładowarki.

- **Auto Standby**: Włącz lub wyłącz automatyczny tryb gotowości w celu oszczędzania energii.
