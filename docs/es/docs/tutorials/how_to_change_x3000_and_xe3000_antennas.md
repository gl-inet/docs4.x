# Cómo instalar o cambiar las antenas externas para Spitz AX o Puli AX

Cuando configure su Spitz AX (GL-X3000) o Puli AX (GL-XE3000) por primera vez, tendrá que instalar las seis antenas externas incluidas en el paquete. Estas antenas externas son:
* 2 antenas Wi-Fi, etiquetadas como "Wi-Fi 6"
* 4 antenas 5G NR, etiquetadas como "5G NR"

Consulte las secciones siguientes para aprender a instalar o cambiar las antenas.

## Instalar antenas externas

Para instalar las antenas, siga estos pasos:

1. Conecte las antenas, sin importar el orden:
    * Conecte las dos antenas Wi-Fi 6 idénticas a los puertos **Wi-Fi 1** y **Wi-Fi 2**. No se requiere un emparejamiento específico.
    * Conecte las cuatro antenas 5G NR idénticas a los puertos **MAIN**, **DIV**, **MIMO 1** y **MIMO2**. No se requiere un emparejamiento específico.
    ![Antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x3000-antennas.jpg){class="gl-80-desktop glboxshadow"}
2. Coloque las antenas apuntando hacia arriba para obtener el mejor rendimiento.

## Cambiar antenas externas

Puede comprar antenas Wi-Fi o 5G NR adicionales para sustituir las incluidas en el paquete. Esto permite personalizar la configuración de red.
Sin embargo, se recomienda usar las antenas originales, ya que se han probado varias veces antes del envío para garantizar una compatibilidad óptima.
Si prefiere usar otras antenas, consulte las secciones siguientes para asegurarse de comprar antenas con los conectores adecuados.

### Antenas Wi-Fi

La base de la antena Wi-Fi utiliza un **conector RP-SMA hembra** con rosca exterior y pin interior. Requiere una antena Wi-Fi que utilice un **conector RP-SMA macho** con rosca interior y hueco interior.
![rp-sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/rp-sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### Antenas 5G NR

La base de la antena 5G NR utiliza un **conector SMA hembra** con rosca exterior y hueco interior. Requiere una antena 5G NR que utilice un **conector SMA macho** con rosca interior y pin interior.
![sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### Solución de problemas

Si compró antenas incorrectas, puede plantearse adquirir un adaptador que convierta entre conectores, por ejemplo, SMA-RP macho a SMA hembra.
