# Ajustes del botón selector

Los ajustes del botón selector le permiten asignar funciones específicas al botón físico de selección del router para un acceso y control rápidos, proporcionando accesos directos cómodos para tareas habituales.

Puede personalizar el comportamiento del botón en el panel de administración web.

## Modelos compatibles

## Configuración

En el lado izquierdo del panel de administración web, vaya a **SYSTEM** -> **Toggle Button Settings**.

![toggle button settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/toggle_button_settings/toggle_button_settings.jpg){class="glboxshadow"}

Antes del firmware v4.8, había cuatro opciones que permitían personalizar la función del botón selector.

- No Function
- AdGuard Home
- OpenVPN Client
- Tor
- WireGuard Client

Desde el firmware v4.8, se han añadido más opciones: Repeater, Wi-Fi y LED. Los usuarios pueden personalizar el botón selector según sus necesidades.

- No Function
- Repeater
- Wi-Fi (Main/Guest Wi-Fi)
- VPN
- Tor
- AdGuard Home
- LED

![toggle button 4.8](https://static.gl-inet.com/docs/router/en/4/interface_guide/toggle_button_settings/toggle_button_4.8.png){class="glboxshadow"}

Al aplicar los ajustes, puede decidir si habilitar o deshabilitar inmediatamente la función seleccionada según la posición on/off izquierda o derecha del botón selector.

**Nota**: Después de reiniciar el dispositivo, el sistema aplicará automáticamente el estado de la función según la posición del botón selector.

Por ejemplo, si configura WireGuard Client para que se controle con el botón selector: cuando el selector está en LEFT (ON), WireGuard Client se inicia automáticamente. Cuando el selector está en RIGHT (OFF), WireGuard Client permanece deshabilitado.
