# OpenVPN, TCP o UDP

Il server OpenVPN dei router GL.iNet supporta sia il protocollo TCP sia UDP. Qual è la differenza?

TCP è più affidabile ma più lento. UDP è più veloce ma meno affidabile.

Se hai bisogno di bassa latenza e trasmissione continua dei dati affinché un'applicazione funzioni correttamente, UDP è la scelta migliore. In caso contrario, TCP è un protocollo affidabile per trasferire dati senza perdite durante la trasmissione.

UDP è più adatto per gaming, streaming o servizi VoIP. TCP è più adatto per email, navigazione web e trasferimento di file.

Ti consigliamo di provare prima il protocollo UDP e passare a TCP solo se riscontri instabilità di connessione, perdita di pacchetti o altri problemi legati all'affidabilità.
