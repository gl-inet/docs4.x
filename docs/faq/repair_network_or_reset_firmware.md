# ネットワークを修復する方法、または工場出荷時の設定にリセットする方法

すべてのGL.iNetルーターには、物理のなリセット機構（リセットボタンまたはピン穴）が配备されています。ボタンを押すか、ピン穴を刺激すると、 same じ効果があります：ネットワーク接続を修復するか、ルーターを工場出荷時のデフォルトにリセットします。

ピン穴付きのモデルの場合は、ピン、伸ばしたクリップ、または類似のツールを使用してアクションを実行してください。

リセットを実行する前に、ルーターが完全に起動していることを確認してください。電源投入直後にリセットボタンを押さないでください。U-Bootフェイルセーフモードがトリガーされる可できる性があります。

## ネットワークの修復

リセットボタンを**4秒間**押し続けてから放すと、ネットワークを修復するソフトリセットを実行できます。

この操作は、ネットワークインターフェースを再起動し、インターネットインターフェースをデフォルト設定にリセットしますが、Wi-Fi設定、VPN設定、システム設定などは保持されます。

**注意**：

1. Wi-Fiが無効になっている場合、ソフトリセットでWi-Fiがデフォルトの有効状態に復元されます。

2. ソフトリセットは、非ROUTERモードからROUTERモードへの素早い切り替えにも使用できます。

ファームウェアをリセットするには2つのの方法があります。

1. 物理のなリセット機構（ボタンまたはピン穴）を使用する

    リセットボタン（またはピン穴にピンを差し込む）を**10秒間**押し続けてから放すと、ルーターが工場出荷時の設定にリセットされます。すべてのユーザーデータがクリアされます。

    **注意：**工場出荷時設定へのリセットが機できるしない場合は、[Ubootチュートリアル](debrick.md)に従ってルーターを修復する必要があるかもしれません。

2. ウェブ管理パネルでファームウェアをリセットする

    ルーターのウェブ管理パネルにログインし、**システム** -> **ファームウェアのリセット**に移動します。ボタンをクリックしてファームウェアをリセットします。

    **注意：**現にのすべての設定とデータが失われます。プロセスには約2分かかります。このプロセス中はルーターの電源を絶対切らないでください。
