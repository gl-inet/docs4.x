# Przywracanie ustawień fabrycznych

W lewym panelu bocznym panelu administracyjnego przejdź do **SYSTEM** -> **Reset Firmware**.

Jeśli router działa nieprawidłowo, możesz spróbować rozwiązać problem, resetując oprogramowanie układowe. Kliknij przycisk **Delete All and Reboot**, aby zresetować oprogramowanie.

**Uwaga:** Wszystkie bieżące ustawienia, aplikacje i dane zostaną utracone. Proces zajmie około 2 minut. NIE wyłączaj routera podczas tego procesu.
