# Filtre de contenu

Content Filter fournit une protection intelligente en ligne grâce à une classification basée sur le DPI, en bloquant automatiquement les sites web nuisibles ou malveillants afin de garder votre réseau propre et sécurisé.

**Note** : Cette fonctionnalité n’est actuellement disponible que sur le **GL-MT5000 (Brume 3)**.

---

Sur le côté gauche du panneau d’administration web, allez à **FLOW CONTROL** > **Content Filter**.

Personnalisez le contenu bloqué (par exemple les applications, les domaines ou les adresses IP), puis cliquez sur **Apply**.

![content filter](https://static.gl-inet.com/docs/router/en/4/interface_guide/content_filter/content-filter.png){class="glboxshadow"}

- **Blocked Apps List** : cette section comprend trois catégories présélectionnées : Gambling, Adult et Malware. Lorsqu’elles sont activées, tous les sites web, services ou applications liés à ces trois catégories sont bloqués. Vous pouvez cliquer sur Edit App pour ajouter d’autres catégories selon vos besoins (par exemple Game, Social Media ou Shopping).

- **Blocked Domain / IP List** : cette section vous permet de saisir manuellement des domaines spécifiques (par exemple `google.com`) ou des adresses IP (par exemple `192.168.8.0/24`) afin d’en bloquer l’accès. La liste prend en charge jusqu’à 10000 entrées.
