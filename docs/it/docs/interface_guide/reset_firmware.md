# Reset Firmware

Sul lato sinistro del pannello di amministrazione web, vai su **SYSTEM** -> **Reset Firmware**.

Se il router presenta malfunzionamenti, puoi provare a risolvere il problema reimpostando il firmware. Fai clic sul pulsante **Delete All and Reboot** per reimpostare il firmware.

**Nota:** tutte le impostazioni, le applicazioni e i dati attuali andranno persi. Il processo richiede circa 2 minuti. NON spegnere il router durante questa procedura.
