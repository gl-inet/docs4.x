# ファームウェアアップデート中に「Installing」でスタックする

ルーターがファームウェアアップデート中に「Installing」でスタックする場合は、で下の手順に従って問題を解決してください。

![stuck on installing](https://static.gl-inet.com/docs/router/en/4/faq/troubleshooting/stuck_on_installing/installing.png){class="glboxshadow"}

各手順の後、問題が解決したかどうかを確認します。解決した場合は、残りの手順をスキップできます。

1. ファームウェアアップデートが成功したかどうかを確認します。

    最も初に、Webブラウザを開き、`192.168.8.1`にアクセスしてログインします。

    次に、左上のファームウェアバージョンを確認します。ターゲットバージョンと一致している場合、アップデートは成功です。一致していない場合は、次の手順に進みます。

2. 安定したネットワーク接続を確保してから、ファームウェアを再アップロードします。

3. 別のブラウザ（Edge、Safariなど）を使用してファームウェアを再アップロードします。
