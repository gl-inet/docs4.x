# GL.iNetルーターのファームウェアをダウンロードする方法

ルーターをアップデートしたい場合（またはU-Bootでブリック問題を解決したい場合）、以下の手順でファームウェアファイルをダウンロードします：

1. [GL.iNet firmwareダウンロードセンター](https://dl.gl-inet.com)で、サーチバーからルーターモデルを選択します。
2. 上部のタブを選択します：
    * **安定版**：日常使用の公式リリースで、大多数のユーザーに推奨されます。
    * **ベータ版**（特定のルー書モデルのみ利用可能）：まだ開発中のプレリリースバージョンです。
    * **スナップショット**（特定のルー書モデルのみ利用可能）：最も新のアップデートを含むデイリービルドです。
3. **ダウンロード**ボタンをクリックします。
