# Come recuperare il log dell'app mobile

Se riscontri malfunzionamenti durante l'uso della nostra GL.iNet Mobile App e vuoi recuperare il log dell'app mobile, puoi seguire i passaggi seguenti.

## Vai alla pagina principale dell'app

Fai clic sull'icona **Wheel** nell'angolo in basso a sinistra.

![applog1](https://static.gl-inet.com/docs/router/en/4/tutorials/app_log/Applog1.jpg){class="glboxshadow gl-50-desktop"}

## Tocca **Enable** in APP LOG(BETA)

![applog2](https://static.gl-inet.com/docs/router/en/4/tutorials/app_log/Applog2.jpg){class="glboxshadow gl-50-desktop"}

## Tocca **Show Log-Window**

![applog3](https://static.gl-inet.com/docs/router/en/4/tutorials/app_log/Applog3.jpg){class="glboxshadow gl-50-desktop"}

## Sul lato sinistro comparira' un'icona blu **L**

Quando tocchi l'icona **L**, verra' mostrato il log della funzione che hai appena usato nell'app.

![applog4](https://static.gl-inet.com/docs/router/en/4/tutorials/app_log/Applog4.jpg){class="glboxshadow gl-50-desktop"}
