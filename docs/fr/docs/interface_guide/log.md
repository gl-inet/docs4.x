# Log

Dans la partie gauche du panneau d'administration web, accédez à **SYSTEM** -> **Log**.

La page Log vous permet d'afficher les journaux System, Kernel, Crash, Cloud et Nginx pour l'analyse et le dépannage.

Certains modèles cellulaires prenant en charge l'eSIM fournissent également un journal eSIM.

Voici un exemple avec le GL-X3000.

![log page](https://static.gl-inet.com/docs/router/en/4/interface_guide/log/log.png){class="glboxshadow"}

La fonctionnalité de **filtrage** est disponible sur la page System Log à partir du firmware v4.8. Vous pouvez choisir un niveau ou un module, ou saisir des mots-clés pour rechercher rapidement dans les journaux.

Cliquez sur **Refresh** pour obtenir les informations de journal les plus récentes.

Cliquez sur **Export Log** pour exporter les journaux System, Kernel, Crash et Cloud. Lorsque vous envoyez un retour à GL.iNet, vous pouvez transmettre le fichier de journal exporté au support technique GL.iNet afin d'accélérer l'analyse du problème.
