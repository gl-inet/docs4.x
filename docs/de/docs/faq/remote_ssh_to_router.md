# Wie greife ich per SSH aus der Ferne auf das Terminal eines GL.iNet-Routers zu?

- **Methode 1**: Wenn das Netzwerk Ihres Routers eine [öffentliche IP-Adresse](../tutorials/how_to_check_if_isp_assigns_you_a_public_ip_address.md) hat, richten Sie bitte einen [WireGuard Server](../interface_guide/wireguard_server.md) ein und verwenden Sie WireGuard, um per SSH auf den Router zuzugreifen. Wenn Sie keine öffentliche IP-Adresse haben, probieren Sie die anderen Methoden unten aus.
- **Methode 2**: [Binden Sie Ihr Gerät an GoodCloud](../interface_guide/cloud.md#setup). Danach können Sie sich auf der [GoodCloud-Website](https://www.goodcloud.xyz){target="_blank"} anmelden und [per SSH aus der Ferne auf das Router-Terminal zugreifen](../interface_guide/cloud.md#remote-access-routers-terminal).
- **Methode 3**: Wenn Sie eine SDWAN-Lösung verwenden möchten, empfehlen wir [AstroWarp](https://www.astrowarp.net){target="_blank"}.
- **Methode 4**: Wenn Sie eine Reverse-Proxy-Lösung verwenden möchten, empfehlen wir [AstroRelay](https://www.astrorelay.com){target="_blank"}.
