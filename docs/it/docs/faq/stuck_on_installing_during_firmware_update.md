# Bloccato su "Installing" durante l'aggiornamento del firmware

Se il router rimane bloccato su "Installing" durante un aggiornamento del firmware, segui i passaggi riportati di seguito per risolvere il problema.

![stuck on installing](https://static.gl-inet.com/docs/router/en/4/faq/troubleshooting/stuck_on_installing/installing.png){class="glboxshadow"}

Dopo ogni passaggio, verifica se il problema è stato risolto. In caso affermativo, salta i passaggi restanti.

1. Verifica che l'aggiornamento del firmware sia andato a buon fine.

    Per prima cosa, apri un browser web, accedi a `192.168.8.1` ed effettua il login.

    Poi controlla la versione del firmware nell'angolo in alto a sinistra. Se corrisponde alla versione di destinazione, l'aggiornamento è riuscito. In caso contrario, passa al passaggio successivo.

2. Assicurati che la connessione di rete sia stabile, quindi ricarica il firmware.

3. Usa un browser diverso (ad esempio Edge o Safari) per ricaricare il firmware.
