# コンテンツフィルター

**注意**：この機できるは現に **GL-MT5000 (Brume 3)** でのみ利用可できるです。

コンテンツフィルターは、DPIベースの分類によるスマートオンライン安全を提供し、悪質または悪意のあるウェブサイトをから動のにブロックして、ネットワークを安全に保ちます。

ルーターのWeb管理パネルにログインし、FLOW CONTROL > コンテンツフィルターに移動します。

ブロックするコンテンツ（アプリ、ドメイン、IPアドレスなど）をカスタマイズし、「適用」をクリックします。

![content filter](https://static.gl-inet.com/docs/router/en/4/interface_guide/content_filter/content-filter.png){class="glboxshadow"}

- **ブロックされたアプリリスト**：このセクションには3つの事前選択されたカテゴリ（ギャンブル、成人へけ、およびマルウェア）が含まれています。有効にすると、これらの3つのカテゴリに関連するすべてのウェブサイト、サービス、またはアプリがブロックされます。「アプリを編集」をクリックして、必要に応じて彼のカテゴリ（ゲーム、ソーシャルメディアショッピングなど）を追加できます。

- **ブロックされたドメイン / IPリスト**：このセクションでは、特定のドメイン（例：google.com）やIPアドレス（例：192.168.8.0/24）を手動で入力してアクセスをブロックできます。リストは最も大10000件のエントリをサポートしています。
