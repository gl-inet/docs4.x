# Log

Po lewej stronie panelu administracyjnego WWW przejdź do **SYSTEM** -> **Log**.

Strona **Log** umożliwia przeglądanie logów System, Kernel, Crash, Cloud i Nginx na potrzeby analizy oraz rozwiązywania problemów.

Niektóre modele komórkowe obsługujące eSIM udostępniają również log eSIM.

Poniżej znajduje się przykład dla modelu GL-X3000.

![log page](https://static.gl-inet.com/docs/router/en/4/interface_guide/log/log.png){class="glboxshadow"}

Funkcja filtrowania jest dostępna na stronie System Log od wersji firmware v4.8. Możesz wybrać poziom lub moduł albo wpisać słowa kluczowe, aby szybko wyszukać logi.

Kliknij **Refresh**, aby pobrać najnowsze informacje z logów.

Kliknij **Export Log**, aby wyeksportować informacje z logów System, Kernel, Crash i Cloud. Gdy przesyłasz opinię do GL.iNet, możesz wysłać wyeksportowany plik logu do działu wsparcia technicznego GL.iNet, co przyspieszy analizę problemu.
