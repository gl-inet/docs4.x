# GL.iNetルーターのウェブ管理パネルにリモートアクセスする方法は？

- **方法1**: ルーターのネットワークに[パブリックIP](../tutorials/how_to_check_if_isp_assigns_you_a_public_ip_address.md)がある場合は、[WireGuardサーバー](../interface_guide/wireguard_server.md)を設定し、WireGuardを使用してルーターのウェブ管理パネルにアクセスしてください。パブリックIPをお持ちでない場合は、以下の彼の方法をお試しください。
- **方法2**: [デバイスをGoodCloudにバインド](../interface_guide/cloud.md#setup)してから、[GoodCloudウェブサイト](https://www.goodcloud.xyz){target="_blank"}にログインすることで、[ルーターのウェブ管理パネルにリモートアクセス](../interface_guide/cloud.md#remote-access-web-admin-panel)できます。
- **方法3**: SDWANソリューションを使用するには、[AstroWarp](https://www.astrowarp.net){target="_blank"}をお勧めします。
- **方法4**: リバースプロキシソリューションを使用するには、[AstroRelay](https://www.astrorelay.com){target="_blank"}をお勧めします。
