# Come collegare un router GL.iNet alla parabola Starlink?

Se devi collegare un router GL.iNet alla parabola Starlink, segui i passaggi riportati di seguito.

**Topology**

![starlinkgl](https://static.gl-inet.com/docs/router/en/4/faq/starlink/starlinkgl.jpg){class="glboxshadow"}

1. Collega l'alimentazione del router Starlink alla parabola Starlink.

2. Collega la porta LAN del router Starlink alla porta WAN del router GL.iNet.

3. Collega un dispositivo alla porta LAN o al Wi-Fi del router GL.iNet, quindi configura il router per l'accesso a Internet. [Configurazione iniziale](first_time_setup.md)

    **Nota**: se il router GL.iNet supporta [Network Acceleration](../interface_guide/network_acceleration.md), vai nel pannello di amministrazione web -> **NETWORK** -> **Network Acceleration** e disattivala.

    ![netacc](https://static.gl-inet.com/docs/router/en/4/faq/starlink/netacc.jpg){class="glboxshadow"}

4. Apri l'app Starlink, vai su Settings e abilita **Bypass mode**.
