# Inhaltsfilter

Der Inhaltsfilter bietet intelligente Online-Sicherheit auf Basis einer DPI-gestützten Klassifizierung und blockiert automatisch schädliche oder bösartige Websites, damit Ihr Netzwerk sauber und sicher bleibt.

**Hinweis**: Diese Funktion ist derzeit nur auf **GL-MT5000 (Brume 3)** verfügbar.

---

Gehen Sie auf der linken Seite des Web-Admin-Panels zu **FLOW CONTROL** > **Content Filter**. 

Passen Sie die blockierten Inhalte an, z. B. Apps, Domains oder IP-Adressen, und klicken Sie auf **Apply**.

![content filter](https://static.gl-inet.com/docs/router/en/4/interface_guide/content_filter/content-filter.png){class="glboxshadow"}

- **Blocked Apps List**: Dieser Abschnitt enthält drei vorab ausgewählte Kategorien: Gambling, Adult und Malware. Wenn sie aktiviert sind, werden alle Websites, Dienste oder Apps blockiert, die zu diesen drei Kategorien gehören. Sie können auf Edit App klicken, um bei Bedarf weitere Kategorien hinzuzufügen, z. B. Game, Social Media oder Shopping.

- **Blocked Domain / IP List**: In diesem Abschnitt können Sie bestimmte Domains (z. B. google.com) oder IP-Adressen (z. B. 192.168.8.0/24) manuell eingeben, um den Zugriff darauf zu blockieren. Die Liste unterstützt bis zu 10000 Einträge.
