# Comment récupérer le journal de l’application mobile

Si vous constatez des dysfonctionnements lors de l’utilisation de notre application mobile GL.iNet et souhaitez récupérer le journal de l’application, suivez les étapes ci-dessous.

## Accédez à la page principale de l’application

Cliquez sur l’icône **Wheel** en bas à gauche.

![applog1](https://static.gl-inet.com/docs/router/en/4/tutorials/app_log/Applog1.jpg){class="glboxshadow gl-50-desktop"}

## Appuyez sur **Enable** dans APP LOG(BETA)

![applog2](https://static.gl-inet.com/docs/router/en/4/tutorials/app_log/Applog2.jpg){class="glboxshadow gl-50-desktop"}

## Appuyez sur **Show Log-Window**

![applog3](https://static.gl-inet.com/docs/router/en/4/tutorials/app_log/Applog3.jpg){class="glboxshadow gl-50-desktop"}

## Une icône bleue **L** apparaîtra sur le côté gauche

Lorsque vous appuyez sur l’icône **L**, le journal de la fonction que vous venez d’utiliser dans l’application s’affiche.

![applog4](https://static.gl-inet.com/docs/router/en/4/tutorials/app_log/Applog4.jpg){class="glboxshadow gl-50-desktop"}
