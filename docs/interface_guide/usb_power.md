# USB & Power

**注意**: このページは**GL-E5800 (Mudi 7)**でのみ利用可能です。

Web管理パネルの左側で、**システム** -> **USB & Power**に移動します。

このページでは、USBプロトコル、電源方向、アイドルタイムアウト、スタンバイ動作など、ルーターのUSB関連設定と電源管理を設定できます。

## USB設定

このセクションでは、ルーターのUSBポートの動作をカスタマイズできます。

![usb settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_usb.png){class="glboxshadow"}

- **USB Protocol Switch**: USBポートのUSB 2.0とUSB 3.1プロトコルを切り替えます。

- **Dual Role USB Mode**: ドロップダウンメニューからUSB動作モードを選択します。DeviceまたはHostに設定できます。

    ![usb role](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/usb_role.png){class="glboxshadow"}

- **Power Direction**: ドロップダウンメニューからUSBポートの電源優先度を選択します。Input PriorityまたはOutput Priorityに設定できます。

- **Power Threshold**: USBポートの特定の電力しきい値百分比を設定して、USB操作に必要な最小電力レベルを制御し、バッテリー寿命を保つことができます。

## 電源設定

このセクションでは、電力消費とデバイス動作を最適化できます。

![power settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_power.png){class="glboxshadow"}

- **Wi-Fi Idle Timeout**: Wi-Fiがスタンバイに入るまでのアイドル時間を設定します（10分から2時間の範囲、またはしない）。

- **Ethernet Idle Timeout**: Ethernetがスタンバイに切り替えられるまでのアイドル時間を設定します（10分から2時間の範囲、またはしない）。

- **Power Off Timeout**: アイドル時にルーターが自動的に電源オフになるまでの遅延時間を設定します（1時間から12時間の範囲、またはしない）。

- **Power On with Charger**: このスイッチを切り替えして、充电器に接続した时的-routerが自動的に電源オンするを有効/無効にします。

- **Auto Standby**: このスイッチを切り替えして、節電のための自動スタンバイを有効/無効にします。
