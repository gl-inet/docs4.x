# VPN使用時にイーサネットWANを設定する必要がありますか？

答えはいいえです。VPNクライアントとして設定する場合、GL.iNetルーターのWANソースをからよりに設定できます。LANとWLANはVPNトンネルでカバーされます。

## Wi-Fiリピーターで設定

GL.iNetルーター（VPNクライアントとして設定）をプライマリールーターに無線接続できます。この方法で、GL.iNetルーターはWi-Fiリピーターとして機能します。

![repater](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/repeater.jpg){class="glboxshadow"}

## イーサネットケーブルで設定

GL.iNetルーター（VPNクライアントとして設定）をイーサネットケーブルでプライマリールーターに接続できます。

![ethernet](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/ethernet.jpg){class="glboxshadow"}

## USBテザリングで設定

GL.iNetルーター（VPNクライアントとして設定）をUSBケーブルで電話に接続できます。これはUSBテザリングとも呼ばれます。

![tethering](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/tethering.jpg){class="glboxshadow"}
