# USB y alimentación

**Nota**: Esta página solo está disponible en **GL-E5800 (Mudi 7)**.

En el lado izquierdo del panel de administración web, vaya a **SYSTEM** -> **USB & Power**.

Esta página le permite configurar ajustes relacionados con USB y la gestión de energía del router, como el protocolo USB, la dirección de alimentación, los tiempos de inactividad y el comportamiento en espera.

## Ajustes de USB

Esta sección le permite personalizar el comportamiento de los puertos USB del router.

![usb settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_usb.png){class="glboxshadow"}

- **USB Protocol Switch**: Cambia entre los protocolos USB 2.0 y USB 3.1 para el puerto USB.

- **Dual Role USB Mode**: Seleccione el modo de funcionamiento USB en el menú desplegable. Puede configurarlo como Device o Host.

  ![usb role](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/usb_role.png){class="glboxshadow"}

- **Power Direction**: Elija la prioridad de alimentación del puerto USB en el menú desplegable. Puede establecerla como Input Priority o Output Priority.

- **Power Threshold**: Establezca un porcentaje específico de umbral de energía para el puerto USB con el fin de controlar el nivel mínimo de energía necesario para las operaciones USB y ayudar a conservar la batería.

## Ajustes de energía

Esta sección le permite optimizar el consumo energético y el comportamiento del dispositivo.

![power settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/usb_power/e5800_power.png){class="glboxshadow"}

- **Wi-Fi Idle Timeout**: Establezca el tiempo de inactividad, desde 10 minutos hasta 2 horas, o nunca, tras el cual el Wi-Fi entrará en espera.

- **Ethernet Idle Timeout**: Establezca el tiempo de inactividad, desde 10 minutos hasta 2 horas, o nunca, para que Ethernet pase al modo de espera.

- **Power Off Timeout**: Establezca el retraso, desde 1 hora hasta 12 horas, o nunca, tras el cual el router se apagará automáticamente cuando esté inactivo.

- **Power On with Charger**: Active o desactive esta opción para que el router se encienda automáticamente al conectarlo a un cargador.

- **Auto Standby**: Active o desactive esta opción para habilitar o deshabilitar el modo de espera automático y ahorrar energía.
