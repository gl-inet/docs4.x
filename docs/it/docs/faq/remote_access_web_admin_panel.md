# Come accedere da remoto al pannello di amministrazione web del router GL.iNet?

- **Metodo 1**: se la rete del router dispone di un [IP pubblico](../tutorials/how_to_check_if_isp_assigns_you_a_public_ip_address.md), configura un [WireGuard Server](../interface_guide/wireguard_server.md) e usa WireGuard per accedere al pannello di amministrazione web del router. Se non hai un IP pubblico, prova gli altri metodi riportati di seguito.
- **Metodo 2**: [associa il dispositivo a GoodCloud](../interface_guide/cloud.md#setup), quindi potrai [accedere da remoto al pannello di amministrazione web del router](../interface_guide/cloud.md#remote-access-web-admin-panel) effettuando l'accesso al [sito web GoodCloud](https://www.goodcloud.xyz){target="_blank"}.
- **Metodo 3**: per usare una soluzione SDWAN, suggeriamo [AstroWarp](https://www.astrowarp.net){target="_blank"}.
- **Metodo 4**: per usare una soluzione reverse proxy, suggeriamo [AstroRelay](https://www.astrorelay.com){target="_blank"}.
