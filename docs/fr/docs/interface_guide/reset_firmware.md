# Réinitialisation du micrologiciel

Dans la partie gauche du panneau d'administration web, accédez à **SYSTEM** -> **Reset Firmware**.

Si le routeur dysfonctionne, vous pouvez tenter de résoudre le problème en réinitialisant le micrologiciel. Cliquez sur le bouton **Delete All and Reboot** pour réinitialiser le micrologiciel.

**Remarque :** Tous vos paramètres actuels, applications et données seront perdus. Le processus prendra environ 2 minutes. NE coupez PAS l'alimentation du routeur pendant ce processus.

![reset firmware](https://static.gl-inet.com/docs/router/en/4/tutorials/reset_firmware/reset_firmware_4.8.png){class="glboxshadow"}
