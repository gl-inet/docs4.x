# Advanced Settings

En el lado izquierdo del panel de administración web, vaya a **SYSTEM** -> **Advanced Settings**.

Puede modificar ajustes avanzados con LuCI, la interfaz web de usuario predeterminada de OpenWrt.

![advanced settings](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/go_to_luci.png){class="glboxshadow"}

Como proyecto abierto e independiente mantenido por OpenWrt, LuCI se ofrece tal cual. GL.iNet no es responsable del mantenimiento de LuCI.

Haga clic en el botón **Go To LuCI** y será redirigido a la página de inicio de sesión de LuCI.

![luci login](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/luci_login.jpg){class="glboxshadow"}

Introduzca la contraseña de inicio de sesión, que es la misma que la del panel de administración web.

Iniciará sesión en la página de LuCI. Este es un ejemplo de la página de LuCI para GL-AX1800, Flint.

![luci logged in](https://static.gl-inet.com/docs/router/en/4/interface_guide/advanced_settings/luci_logged_in.jpg){class="glboxshadow"}
