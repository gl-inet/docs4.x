# Come installare o sostituire le antenne esterne dei router cellulari?

Questo tutorial presenta l'installazione e la sostituzione delle antenne esterne per alcuni router cellulari.

## Installare le antenne esterne

### Spitz AX / Puli AX

Spitz AX (GL-X3000) e Puli AX (GL-XE3000) includono le seguenti antenne esterne:

* 2 antenne Wi-Fi (RP-SMA, 2.4G~2.5GHz, 5.15~5.84GHz)
* 4 antenne 5G NR (SMA, 600MHz~6GHz)

Segui questi passaggi per installare le antenne esterne.

1. Collega le antenne, in qualsiasi ordine.

    * Collega le due antenne Wi-Fi 6 identiche alle porte **Wi-Fi 1** e **Wi-Fi 2**.
    * Collega le quattro antenne 5G NR identiche a **MAIN**, **DIV**, **MIMO 1** e **MIMO2**.

    ![x3000 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x3000-antennas.jpg){class="gl-80-desktop glboxshadow"}

2. Posiziona le antenne verso l'alto per ottenere prestazioni ottimali.

### Spitz Plus

Spitz Plus (GL-X2000) include le seguenti antenne esterne:

* 2 antenne Wi-Fi (RP-SMA, 2.4G~2.5GHz, 5.15~5.84GHz)
* 2 antenne 4G (SMA, 600MHz~3.8GHz)

Segui questi passaggi per installare le antenne esterne.

1. Collega le antenne, in qualsiasi ordine.

    - Collega le due antenne Wi-Fi identiche alle porte **Wi-Fi 6**.
    - Collega le due antenne 4G identiche alla porta **4G**.

    ![x2000 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x2000-back.png){class="gl-80-desktop glboxshadow"}

2. Posiziona le antenne verso l'alto per ottenere prestazioni ottimali.

### Spitz

Spitz (GL-X750/GL-X750V2) include 2 antenne esterne 4G (SMA, 700M~2700MHz).

Segui questi passaggi per installare le antenne esterne.

1. Collega le due antenne 4G identiche alle porte antenna.

    ![x750v2 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x750v2_back.png){class="gl-80-desktop glboxshadow"}

2. Posiziona le antenne verso l'alto per ottenere prestazioni ottimali.

### Collie

Collie (GL-X300B-RS485) include le seguenti antenne:

* 1 antenna esterna 4G
* 2 antenne esterne Wi-Fi

Segui questi passaggi per installare le antenne.

1. Collega le antenne, in qualsiasi ordine.

    - Collega le due antenne Wi-Fi identiche alle porte **2.4G**.
    - Collega l'unica antenna 4G alla porta **4G**.

    ![x300b antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x300b-rs485.png){class="gl-80-desktop glboxshadow"}

    ![x300b antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x300b-rs485-back.png){class="gl-80-desktop glboxshadow"}

2. Posiziona le antenne verso l'alto per ottenere prestazioni ottimali.

## Sostituire le antenne esterne

Le indicazioni seguenti si applicano a tutti i router cellulari con interfacce per antenne esterne, eccetto Mudi 7 (GL-E5800).

Puoi acquistare antenne Wi-Fi o 4G/5G NR aggiuntive per sostituire quelle incluse nella confezione e personalizzare cosi' la configurazione di rete.

Tuttavia, consigliamo di usare le antenne originali, poiche' sono state sottoposte a test approfonditi prima della spedizione per garantire prestazioni e compatibilita' ottimali.

Se preferisci usare antenne di terze parti, fai riferimento alle sezioni seguenti per assicurarti di scegliere antenne con il tipo corretto di connettore.

**Suggerimento**: se hai acquistato antenne con connettori errati, puoi valutare l'acquisto di un adattatore che converta tra connettori diversi, ad esempio da SMA-RP maschio a SMA femmina.

### Antenne Wi-Fi

La base dell'antenna Wi-Fi usa un **connettore RP-SMA femmina** (filettatura esterna e pin interno). E' quindi necessaria un'antenna Wi-Fi con **connettore RP-SMA maschio** (filettatura interna e foro interno).

![rp-sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/rp-sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### Antenne 5G NR

La base dell'antenna 5G NR usa un **connettore SMA femmina** (filettatura esterna e foro interno). E' quindi necessaria un'antenna 5G NR con **connettore SMA maschio** (filettatura interna e pin interno).

![sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/sma-connectors.jpg){class="gl-80-desktop glboxshadow"}
