# Strefa czasowa

W lewym panelu bocznym interfejsu administracyjnego przejdź do **SYSTEM** -> **Time Zone**.

Router rejestruje znaczniki czasu zdarzeń zgodnie z czasem systemowym, a niektóre funkcje wymagają prawidłowego czasu do poprawnego działania. Upewnij się, że wybrana strefa czasowa jest właściwa i zsynchronizowana.

Strefa czasowa nie jest synchronizowana automatycznie. Kliknij przycisk **Sync**, aby ją zaktualizować.
