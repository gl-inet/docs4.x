# Tethering iPhone'a nie działa

Jeśli tethering iPhone'a nie działa, wypróbuj poniższe kroki, aby rozwiązać problem.

1. Upewnij się, że opcja **Maximise Compatibility** jest ustawiona na **Enabled** (iPhone 8 lub nowszy).

    ![maxcompat](https://static.gl-inet.com/docs/router/en/4/tutorials/iphone_tethering_fail/maxcompat.PNG){class="glboxshadow gl-50-desktop" }

2. Wyłącz hotspot w telefonie, odczekaj 5 sekund, a następnie włącz go ponownie.

    ![hotspot](https://static.gl-inet.com/docs/router/en/4/tutorials/iphone_tethering_fail/hotspot.PNG){class="glboxshadow gl-50-desktop"}

3. Upewnij się, że wersja iOS to 15 lub nowsza.

4. Odłącz i ponownie podłącz kabel Lightning / Type-C.

5. Użyj innego kabla USB. Upewnij się, że obsługuje przesyłanie danych, a nie tylko ładowanie.

6. Użyj oryginalnego zasilacza do routera.

7. Uruchom ponownie router i iPhone'a. Czasami ponowne uruchomienie iPhone'a rozwiązuje problem z tetheringiem.
