# ルーターのファームウェアを手動でアップグレードまたはダウングレードする方法（v4.x から v4.x）

このチュートリアルでは、**ルーターのファームウェアを手動でアップグレードまたはダウングレードする方法（v4.x から v4.x）** を説明します。手動アップグレードと手動ダウングレードの手順は同じです。

!!! Note "ルーターのファームウェアのアップグレードとダウングレードについて"

    **Upgrade:** ファームウェアバージョン 4.x を実行しているGL.iNetルーターでは、自動更新機能は提供されていません。

    新しいファームウェアバージョンが利用可能になると、ルーターの管理画面にサインインした後に "Upgrade Reminder" というメッセージが表示されます。そこに表示される最新のファームウェアバージョンをインストールするには、**Upgrade** ボタンをクリックします。特定のファームウェアバージョンへアップグレードしたい場合は、以下の手順に従ってルーターを手動でアップグレードしてください。

    **Downgrade:** 特定の問題を解決するために、ルーターのファームウェアをダウングレードできます。

## 1. ルーターが最新のファームウェアバージョンを実行しているか確認する（アップグレード時のみ）

1. Webブラウザーでルーターの管理画面URL（例: `192.168.8.1`）を入力し、サインインします。
2. 左側のサイドバーで **SYSTEM** > **Upgrade** を選択します。

## 2. ファームウェアファイルをダウンロードする

1. [firmware download center](https://dl.gl-inet.com/) の検索バーでルーターモデルを検索して選択します。
2. **Stable** タブまたは他のタブで、ダウンロードしたいファームウェアバージョンの横にある **Download for common upgrade and uboot** を選択します。

## 3. ファームウェアをアップロードする

以下の手順は、ルーターの管理画面からファームウェアをアップロードする場合の説明です。（GL.iNet mobile app からファームウェアをアップロードする場合は、[アプリをダウンロード](https://www.gl-inet.com/app/)して設定してください。）

1. Webブラウザーでルーターの管理画面URL（例: `192.168.8.1`）を入力し、サインインします。
2. （任意）現在の設定をバックアップしたい場合は、以下の手順に従ってください。

    ??? "現在の設定をバックアップする"

        a. 左側のサイドバーで **SYSTEM** > **Advanced Settings** をクリックします。

        b. リンクまたは **Go To LuCI** ボタンをクリックして、LuCIのログインページを開きます。

        c. 管理者パスワードを入力し、**Log in** をクリックします。

        d. **System** > **Backup / Flash Firmware** をクリックします。

        e. **Backup** の下にある **Generate archive** をクリックします。現在の設定を含むファイルがデバイスにダウンロードされます。

        **このファイルは、バックアップ時点のファームウェアバージョンでのみ使用でき、他のファームウェアバージョンでは使用できません。**

3. 左側のサイドバーで **SYSTEM** > **Upgrade** をクリックします。
4. **Local Upgrade** をクリックし、先ほどダウンロードしたファイルを選択します。
5. 現在の設定（たとえばルーターの管理者パスワード）を保持するには、**Keep Settings** のスイッチをオンにします。
6. **Install** をクリックします。

**Note:** アップグレード中は、ルーターの電源を切らないでください。アップグレードが完了すると、ルーターのログイン画面が表示されます。

ファームウェア更新中にルーター設定が失われた場合は、ルーター設定を復元してください。

上記の方法でうまくいかない場合は、[U-boot](../faq/debrick.md) を使ってファームウェアをアップグレードしてみてください。
