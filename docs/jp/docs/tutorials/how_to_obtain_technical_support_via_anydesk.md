# AnyDesk で技術サポートを受ける方法

**リモートサポートでは、当社の技術担当者が直接ネットワーク設定を確認し、ネットワーク構成の支援やトラブルシューティングを迅速に行えます。**

**リモートサポートを受ける前に、以下の準備に数分お時間をください:**

## 1. 不要なアプリケーションを閉じる

事前に不要なアプリケーションを閉じてください。気が散るのを防ぎ、プライバシーを保護できます。

## 2. 安定したネットワークを確保する

リモートサポート中の切断や遅延を防ぐため、可能であればコンピューターでは Wi-Fi と有線 LAN の両方を利用できる状態にしておくことを推奨します。

## 3. AnyDeskをダウンロードしてインストール

[AnyDesk公式ウェブサイト](https://anydesk.com/)にアクセスして、オペレーティングシステムに応じたバージョンをダウンロードします。

![download anydesk](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/install_anydesk.png){class="glboxshadow"}

インストール後にAnyDeskを起動します。

## 4. AnyDesk アドレスを確認する

AnyDesk を起動すると、数字のアドレスが表示されます（通常は「123 456 789」の形式です）。これがサポート担当者へ共有するアドレスです。

![get anydesk address](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/anydesk_address.png){class="glboxshadow"}

## 5. AnyDesk アドレスを共有する

サポート担当者へ AnyDesk アドレスを送信し、デバイスへ接続できるようにします。

## 6. 接続リクエストを受け入れる

技術者が接続を試みると、接続リクエストのプロンプトが表示されます。

接続しようとしている技術担当者が信頼できることを確認したうえで、`Accept` をクリックしてデバイスへのアクセスを許可してください。

![accept request](https://static.gl-inet.com/docs/router/en/4/tutorials/remote_support_via_anydesk/accept_request.png){class="glboxshadow"}

## 7. セッション中に連絡を取る

サポートセッション中は、スピーカー、AnyDesk セッション内のチャットウィンドウ、またはその他の方法で随時コミュニケーションを取り、問題が適切に解決されるようにしてください。

## 8. セッションを終了する

サポートセッションが終了したら、AnyDesk 画面内の赤い `End Session` ボタンをクリックして接続を終了します。これにより、すべてのリモートアクセス権限が解除されます。プライバシー保護を徹底するため、あわせて以下も行ってください:

- AnyDesk アプリケーションを完全に終了する（最小化するだけではなく終了する）。
- 残留プロセスについて懸念がある場合は、コンピュータを再起動します。
- リモートアクセス権限が有効なまま残っていないことを確認するため、システム設定も確認する。
