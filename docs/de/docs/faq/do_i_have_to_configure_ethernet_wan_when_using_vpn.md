# Muss ich Ethernet-WAN konfigurieren, wenn ich VPN verwende?

Die Antwort lautet: Nein. Sie können jede beliebige WAN-Quelle für Ihren GL.iNet Router konfigurieren, wenn Sie ihn als VPN-Client einrichten. Ihr LAN und WLAN werden durch den VPN-Tunnel abgedeckt.

## Über Wi-Fi Repeater konfigurieren

Sie können den GL.iNet Router (als VPN-Client eingerichtet) drahtlos mit Ihrem primären Router verbinden. Auf diese Weise funktioniert der GL.iNet Router als Wi-Fi Repeater.

![repater](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/repeater.jpg){class="glboxshadow"}

## Über Ethernet-Kabel konfigurieren

Sie können den GL.iNet Router (als VPN-Client eingerichtet) über ein Ethernet-Kabel mit Ihrem primären Router verbinden.

![ethernet](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/ethernet.jpg){class="glboxshadow"}

## Über USB Tethering konfigurieren

Sie können den GL.iNet Router (als VPN-Client eingerichtet) über ein USB-Kabel mit Ihrem Telefon verbinden. Dies wird auch als USB Tethering bezeichnet.

![tethering](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/tethering.jpg){class="glboxshadow"}
