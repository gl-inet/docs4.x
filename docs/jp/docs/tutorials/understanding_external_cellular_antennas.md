# 外部セルラーアンテナについて

外部セルラーアンテナは、状況によってセルラー接続の品質を改善できる場合があります。ただし、ダウンロード速度やアップロード速度の向上を保証するものではありません。

このガイドでは、外部アンテナが役立つケース、互換性のあるアンテナの選び方、セルラー性能に影響する要因について説明します。

## 外部アンテナはどのような場合に使用すべきですか？

ルーターが、セルラー信号がデバイスに届く前に弱くなる場所に設置されている場合、外部アンテナで接続が改善することがあります。

代表的な例は次のとおりです。

- RV やキャンピングカー
- 車両
- 地下室
- コンクリート、レンガ、金属の壁がある建物
- ユーティリティルームやネットワークキャビネット
- セルラーのカバレッジが弱い地方エリア

アンテナを購入する前に、ルーターを窓の近くやより高い場所へ移動してみてください。多くの場合、ルーターの設置場所を変えるだけで信号品質が大きく改善します。

## 外部アンテナが役立たないのはどのような場合ですか？

外部アンテナですべてのセルラー接続問題を解決できるわけではありません。

次の場合は、改善がほとんどない、またはまったくないことがあります。

- ルーターのセルラー信号がすでに強い。
- セルラー基地局が混雑している。
- モバイルキャリアがネットワーク速度を制限している。
- アンテナがキャリアの周波数帯に対応していない。
- 長いアンテナケーブルによって過度な信号損失が発生している。
- アンテナがルーターの内蔵アンテナとほぼ同じ場所に設置されている。

信号が強くなっても、必ずしもインターネット速度が速くなるわけではありません。

## 適切なアンテナを選ぶ

アンテナを購入する前に、ルーターとモバイルキャリアの両方に対応していることを確認してください。

次の点を確認します。

- コネクタタイプがルーターと一致している。
- アンテナがキャリアで使用されるセルラー周波数帯に対応している。
- ルーターに応じて、アンテナが LTE、5G、またはその両方に対応している。
- アンテナが設置環境（屋内、車両、屋外など）に適している。

正しいアンテナコネクタタイプについては、ルーターの製品ドキュメントを参照してください。

## MIMO について

現在の多くの LTE および 5G ネットワークでは、スループットと接続の信頼性を向上させるために **Multiple Input Multiple Output (MIMO)** 技術が使用されています。

多くの GL.iNet セルラールーターには、MIMO に対応するため複数のセルラーアンテナポートがあります。

最適な性能を得るには、次の点に従ってください。

- 必要なアンテナリードをすべて接続します。
- 可能な場合は互換性のある MIMO アンテナを使用します。
- アンテナの間隔と向きについて、アンテナメーカーの推奨事項に従います。

複数のアンテナ用に設計されたルーターでアンテナを 1 本だけ使用すると、全体的な性能が低下することがあります。

## アンテナ利得とケーブル損失

アンテナ利得は通常 **dBi** で表されます。

利得が高いほど性能が良いとは限りません。

高利得アンテナは通常、より指向性が強くなるため、接続先のセルラー基地局が分かっている固定設置に適しています。

ケーブル長も同じく重要です。アンテナケーブルは、信号がルーターに到達する前に信号損失を発生させます。

最良の結果を得るには、次の点に従ってください。

- 実用上できるだけ短いケーブルを使用します。
- 不要なアダプターを避けます。
- 長いケーブル配線では低損失の同軸ケーブルを使用します。

## 無指向性アンテナと指向性アンテナ

### 無指向性アンテナ

無指向性アンテナは、すべての方向から信号を受信します。

一般的には次の用途に推奨されます。

- 車両
- RV
- 移動設置
- 近くに複数の基地局があるエリア

### 指向性アンテナ

指向性アンテナは、特定の方向に向けて受信を集中させます。

一般的には次の場合に推奨されます。

- ルーターを 1 か所に固定して使用する。
- 接続先のセルラー基地局が分かっている。
- セルラーのカバレッジが弱い。
- 最大限の信号品質が必要である。

指向性アンテナで最適な性能を得るには、通常、慎重な向き調整が必要です。

## 改善効果を測定する

ルーター画面に表示される信号バーだけに頼らないでください。

アンテナの設置前後で信号測定値を比較してください。

有用な測定値は次のとおりです。

- RSRP（信号強度）
- RSRQ（信号品質）
- SINR（信号対干渉比）
- 接続中の LTE または 5G バンド
- キャリアアグリゲーションの状態

セルラー性能は 1 日の中で自然に変動するため、同じような条件で複数回の速度テストを実行してください。

## よくある質問

### どのセルラーアンテナでも使用できますか？

いいえ。

アンテナは正しいコネクタタイプを使用し、ルーターとモバイルキャリアで使われるセルラー周波数帯に対応している必要があります。

### 最も高利得のアンテナを購入すべきですか？

必ずしもそうではありません。

公称利得だけで選ぶのではなく、設置環境に適したアンテナを選んでください。

### 外部アンテナを使うと必ずインターネット速度は上がりますか？

いいえ。

性能は、信号品質、キャリアアグリゲーション、基地局の混雑、ネットワーク状況、モバイルキャリアなど、多くの要因に左右されます。

### 外部アンテナを接続した場合、内蔵セルラーアンテナも引き続き使われますか？

いいえ。

外部セルラーアンテナポートを備えた GL.iNet セルラールーターでは、互換性のある外部アンテナを接続すると、対応する内蔵セルラーアンテナが無効になります。

最適な性能を得るには、必要な外部アンテナリードをすべて接続してください。複数のセルラーアンテナ用に設計されたルーターに 1 本だけ接続すると、MIMO 性能と全体的な接続品質が低下することがあります。

### Wi-Fi アンテナをセルラーアンテナポートに接続できますか？

いいえ。

Wi-Fi アンテナとセルラーアンテナは異なる無線システム向けに設計されており、互換性はありません。
