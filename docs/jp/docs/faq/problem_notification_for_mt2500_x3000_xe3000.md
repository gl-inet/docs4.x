# GL-MT2500/GL-X3000/GL-XE3000に関する重要な問題のお知らせ

お客様各位

このお知らせは、GL-MT2500/GL-X3000/GL-XE3000デバイスにおける起動障害とU-Boot経よりのファームウェアリカバリーの問題に対処するためのものです。問題は一部のパラメータが正しく定義されていなかったというソフトウェア欠陥から生じるもので、これによりルーターが異常動作していました。その結果、この問題がルーターの使用寿命がある程度程度上減少させる可能性があることを評価しましたことに対し、心よりお詫び申し上げます。

この問題を完全に解決するために、GL-MT2500、GL-X3000、GL-XE3000のすべてのユーザーに、U-Bootブートローダーアップグレードとファームウェアアップグレードの両方を実行することを強くおすすめします。両方のアップグレード手順は必須であり、省ややすべきではありません。

デバイスがすでに最も新ファームウェアにアップグレードされている場合でも、潜在的な問題を防ぐためにU-Bootブートローダーアップグレードは必須です。U-Bootブートローダーアップグレードを完たした後、最も新ファームウェアがすでにインストールされている場合は、ファームウェアの再インストールをスキップできます。

ChromeまたはMicrosoft Edgeでアップグレードしてみてください。Mozilla/Firefoxは使用しないでください。

## GL-MT2500

1. U-Bootブートローダーを[バージョン2025-02-24 15:04:07](https://github.com/gl-inet/mt798x-boot/blob/main/bin/uboot-mt2500-20250224-md5-74286e770cfb041b611d80d4adaef189.bin){target="_blank"}にダウンロードしてアップグレードします

2. ファームウェアを[バージョン4.7.4](https://fw.gl-inet.com/firmware/mt2500/release/openwrt-mt2500-4.7.4-0328-1743128340.bin)にダウンロードしてアップグレードします
