# プラグイン

Web Admin Panel の左側で、**APPLICATIONS** -> **Plug-ins** に移動します。

このページでは OpenWrt パッケージを管理できます。必要に応じてパッケージのインストールや削除を行ってください。

サードパーティ製プラグインを使用する前に、**Update** ボタンをクリックすることをお勧めします。ただし、`nginx`、`adguard home`、`tailscale` など、GL.iNet 公式ファームウェアにプリインストールされているプラグインは更新しないでください。

![Plug-ins](https://static.gl-inet.com/docs/router/en/4/tutorials/plug-ins/plug-ins.png){class="glboxshadow"}

ユーザーが手動でインストールしたサードパーティ製プラグインは更新できますが、それらに起因するセキュリティ上の問題について GL.iNet は責任を負いません。お問い合わせは各プラグインの提供元へお願いします。
