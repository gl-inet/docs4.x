## Router celular

### GL-XE3000

![gl-xe3000 led](https://static.gl-inet.com/docs/router/en/4/faq/led/xe3000led.jpg){class="glboxshadow gl-70-desktop"}

- LED de alimentación
  - Verde: encendido
  - Apagado: apagado

- LED de batería, durante la carga
  - Rojo parpadeando: nivel de batería 0%-20%
  - Violeta parpadeando: nivel de batería 21%-50%
  - Verde parpadeando: nivel de batería 51%-99%
  - Verde fijo: nivel de batería 100%

- LED de batería, sin carga

  Cuando no se está cargando, el LED de batería se ilumina durante 5 segundos después de pulsar el botón de encendido.
  - Rojo: nivel de batería 0%-20%
  - Violeta: nivel de batería 21%-50%
  - Verde: nivel de batería 51%-100%

- LED de Internet
  - Encendido: Internet conectado
  - Apagado: Internet desconectado

- LED de Wi-Fi 2.4G
  - Encendido: Wi-Fi 2.4G listo
  - Parpadeando: Wi-Fi 2.4G en comunicación

- LED de Wi-Fi 5G
  - Encendido: Wi-Fi 5G listo
  - Parpadeando: Wi-Fi 5G en comunicación

- LED celular
  - Torre móvil conectada

## Puertas de enlace de seguridad y mini routers

### GL-MT5000

![gl-mt5000 interface](https://static.gl-inet.com/docs/router/en/4/faq/led/mt5000_interface.png){class="glboxshadow"}

- LED del sistema
  - Azul: inicio del sistema
  - Azul parpadeando: buscando red
  - Blanco: funcionamiento normal
- LED VPN
  - Apagado: el cliente VPN no está conectado
  - Parpadeando: el cliente VPN se está conectando
  - Fijo: el cliente VPN está conectado

Nota: El estado del servidor VPN no se muestra en el LED.

### GL-MT2500/GL-MT2500A

![gl-mt2500 interface](https://static.gl-inet.com/docs/router/en/4/faq/led/mt2500_interface.jpg){class="glboxshadow"}

  - Blanco: funcionamiento normal

  - Parpadeando: el cliente VPN se está conectando
  - Fijo: el cliente VPN está conectado

### GL-MT300N-V2

![gl-mt300n-v2 interface](https://static.gl-inet.com/docs/router/en/4/faq/led/mt300n-v2_interface.png){class="glboxshadow"}

- LED de alimentación
  - Encendido: encendido
  - Apagado: apagado

- LED central, configurable
  - Encendido: este LED ha sido personalizado
  - Apagado: este LED no ha sido personalizado, modo predeterminado

- LED inalámbrico
  - Encendido: Wi-Fi 2.4G conectado
  - Apagado: Wi-Fi 2.4G desconectado

  En lotes anteriores, el LED inalámbrico puede mostrarse en rojo en lugar de blanco durante el funcionamiento normal.
