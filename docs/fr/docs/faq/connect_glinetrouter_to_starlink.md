# Comment connecter un routeur GL.iNet à une antenne Starlink ?

Si vous devez connecter un routeur GL.iNet à une antenne Starlink, suivez les étapes ci-dessous.

**Topologie**

![starlinkgl](https://static.gl-inet.com/docs/router/en/4/faq/starlink/starlinkgl.jpg){class="glboxshadow"}

1. Branchez l'alimentation de votre routeur Starlink à l'antenne Starlink.

2. Connectez le port LAN du routeur Starlink au port WAN du routeur GL.iNet.

3. Connectez un appareil au port LAN ou au Wi-Fi du routeur GL.iNet, puis configurez le routeur pour l'accès à Internet. [Première configuration](first_time_setup.md)

    **Remarque** : si votre routeur GL.iNet prend en charge la [Network Acceleration](../interface_guide/network_acceleration.md), allez dans le panneau d'administration web -> **NETWORK** -> **Network Acceleration** et désactivez-la.

    ![netacc](https://static.gl-inet.com/docs/router/en/4/faq/starlink/netacc.jpg){class="glboxshadow"}

4. Ouvrez l'application Starlink, allez dans Settings et activez le **Bypass mode**.
