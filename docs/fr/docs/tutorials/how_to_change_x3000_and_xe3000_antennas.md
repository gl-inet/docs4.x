# Comment installer ou remplacer les antennes externes du Spitz AX ou du Puli AX

Lorsque vous configurez votre Spitz AX (GL-X3000) ou votre Puli AX (GL-XE3000) pour la première fois, vous devez installer les six antennes externes incluses dans l'emballage. Ces antennes externes sont : 

* 2 x antennes Wi-Fi (étiquetées "Wi-Fi 6")
* 4 x antennes 5G NR (étiquetées "5G NR")

Consultez les sections ci-dessous pour savoir comment installer ou remplacer les antennes. 

## Installer des antennes externes

Pour installer les antennes, suivez les étapes suivantes : 

1. Fixez les antennes (dans n'importe quel ordre) : 
    * Fixez les deux antennes Wi-Fi 6 identiques aux ports **Wi-Fi 1** et **Wi-Fi 2** (aucun appairage spécifique n'est nécessaire).
    * Fixez les quatre antennes 5G NR identiques aux ports **MAIN**, **DIV**, **MIMO 1** et **MIMO2** (aucun appairage spécifique n'est nécessaire).

    ![Antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x3000-antennas.jpg){class="gl-80-desktop glboxshadow"}

2. Orientez les antennes vers le haut pour des performances optimales. 

## Remplacer des antennes externes 

Vous pouvez acheter des antennes Wi-Fi ou 5G NR supplémentaires pour remplacer celles incluses dans l'emballage. Cela permet aux utilisateurs de personnaliser leurs paramètres réseau.

Il est cependant recommandé d'utiliser les antennes d'origine, car elles ont été testées à plusieurs reprises avant l'expédition afin de garantir une compatibilité optimale.

Si vous préférez utiliser d'autres antennes, reportez-vous aux sections ci-dessous pour vous assurer d'acheter des antennes avec les connecteurs appropriés. 

### Antennes Wi-Fi

La base de l'antenne Wi-Fi utilise un **connecteur RP-SMA femelle** (filetage externe et broche interne). Elle nécessite une antenne Wi-Fi utilisant un **connecteur RP-SMA mâle** (filetage interne et trou interne).

![rp-sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/rp-sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### Antennes 5G NR

La base de l'antenne 5G NR utilise un **connecteur SMA femelle** (filetage externe et trou interne). Elle nécessite une antenne 5G NR utilisant un **connecteur SMA mâle** (filetage interne et broche interne).

![sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### Dépannage

Si vous avez acheté des antennes incorrectes, vous pouvez envisager d'acheter un adaptateur qui convertit les connecteurs (par exemple, SMA-RP male vers SMA female).
