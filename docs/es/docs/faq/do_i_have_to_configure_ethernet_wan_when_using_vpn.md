# ¿Tengo que configurar Ethernet WAN al usar VPN?

La respuesta es no. Puede configurar cualquier fuente WAN para su router GL.iNet cuando lo configure como cliente VPN. Su LAN y WLAN quedarán cubiertas por el túnel VPN.

## Configuración mediante repetidor Wi-Fi

Puede conectar el router GL.iNet, configurado como cliente VPN, a su router principal de forma inalámbrica. De este modo, el router GL.iNet funcionará como un repetidor Wi-Fi.

![repater](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/repeater.jpg){class="glboxshadow"}

## Configuración mediante cable Ethernet

Puede conectar el router GL.iNet, configurado como cliente VPN, a su router principal mediante un cable Ethernet.

![ethernet](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/ethernet.jpg){class="glboxshadow"}

## Configuración mediante USB Tethering

Puede conectar el router GL.iNet, configurado como cliente VPN, a su teléfono mediante un cable USB, lo que también se conoce como USB Tethering.

![tethering](https://static.gl-inet.com/docs/router/en/4/faq/wan_source_vpn/tethering.jpg){class="glboxshadow"}
