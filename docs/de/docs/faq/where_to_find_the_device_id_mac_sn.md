# Wo finde ich die Device ID, MAC-Adresse und Seriennummer?

## Methode 1

Diese Informationen finden Sie auf der Unterseite Ihres Routers.

Bei neueren Modellen ähnelt das Etikett auf der Unterseite dem untenstehenden Bild und zeigt die grundlegenden Geräteinformationen an. Im folgenden Beispiel ist die MAC-Adresse **E4:95:6E:40:DB:A1**, die Device ID **hh0dba1** und die Seriennummer **7c0be4bb45d96057**.

![label of gl-x750](https://static.gl-inet.com/docs/router/en/4/tutorials/where_to_find_the_device_id_mac_sn/back_label_new.png){class="glboxshadow"}

Bei älteren Modellen gibt es auf dem Etikett an der Unterseite keine Device ID. Stattdessen finden Sie dort die DDNS, und die ersten 7 Zeichen der DDNS sind die Device ID. Im folgenden Beispiel ist die MAC-Adresse **E4:95:6E:40:DB:A1**, die Device ID **hh0dba1** und die Seriennummer **7c0be4bb45d96057**.

![label of gl-ar150](https://static.gl-inet.com/docs/router/en/4/tutorials/where_to_find_the_device_id_mac_sn/back_label_old.png){class="glboxshadow"}

## Methode 2

Melden Sie sich im Web-Admin-Panel an, navigieren Sie zu SYSTEM -> Overview, und dort finden Sie Details zu Ihrem Gerät.

![device overview](https://static.gl-inet.com/docs/router/en/4/tutorials/where_to_find_the_device_id_mac_sn/overview_device_info.png){class="glboxshadow"}
