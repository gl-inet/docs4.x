# Jak podłączyć do Wi-Fi urządzenie obsługujące tylko Ethernet za pomocą routera GL.iNet?

Jeśli Twoje urządzenie obsługuje wyłącznie połączenie Ethernet, ale musi połączyć się z siecią bezprzewodową, router GL.iNet może pomóc w połączeniu obu tych światów.

Router GL.iNet może działać jako most Ethernet-Wi-Fi dla urządzeń, które mają tylko połączenie przewodowe, umożliwiając im dołączenie do sieci bezprzewodowej. Osiąga się to przez skonfigurowanie routera jako repeater.

Podłącz urządzenie do portu LAN routera, a następnie skonfiguruj [repeater](../interface_guide/internet_repeater.md).
