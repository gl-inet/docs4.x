# Internet

Inicie sesión en el panel web de administración y vaya a **INTERNET**.

La mitad superior muestra un diagrama con el estado de Internet, y la mitad inferior enumera cuatro o cinco métodos de acceso a Internet, cuya disponibilidad varía según el modelo.

## Diagrama de estado

![status diagram](https://static.gl-inet.com/docs/router/en/4/tutorials/internet/status_diagram.png){class="glboxshadow"}

**El lado izquierdo** del diagrama muestra el estado actual de Internet.

- Una línea verde continua significa que la conexión de red está activa.
- Una línea gris discontinua significa que no hay conexión de red.
- Una línea blanca continua con un signo de exclamación amarillo significa que hay conexión, pero no hay acceso a Internet.

**La parte central** del diagrama muestra el modelo actual del router y el estado de [AdGuard Home](adguardhome.md), [IPv6](ipv6.md), [VPN](vpn_dashboard.md), [Tor](tor.md), [5G Main Wi-Fi](wireless.md), [2.4G Main Wi-Fi](wireless.md), [5G Guest Wi-Fi](wireless.md) y [2.4G Guest Wi-Fi](wireless.md).

- El color verde indica que la función está habilitada y el gris indica que está deshabilitada. En el caso de VPN y Tor, el color blanco significa que se está conectando.
- Pase el cursor sobre un icono Wi-Fi habilitado para mostrar su código QR Wi-Fi. Puede escanearlo con su teléfono o tableta para conectarse rápidamente a la red Wi-Fi correspondiente.

  ![wifi qr code](https://static.gl-inet.com/docs/router/en/4/tutorials/internet/wifi_qrcode.png){class="glboxshadow"}

**El lado derecho** del diagrama muestra los dispositivos ([Clients](clients.md)) conectados al router.

- Los dispositivos conectados de forma inalámbrica se muestran en la parte superior y los dispositivos cableados en la parte inferior.
- Los números junto a cada sección indican la cantidad de dispositivos conectados.

## Métodos de acceso a Internet

Hay hasta cuatro métodos de acceso a Internet. Consulte los tutoriales correspondientes a continuación o comience con el [Setup Wizard](#setup-wizard).

- [Ethernet](internet_ethernet.md)
- [Repeater](internet_repeater.md)
- [Tethering](internet_tethering.md)
- [Cellular](internet_cellular.md)

## Setup Wizard

El asistente de configuración está disponible desde el firmware v4.7.

En la página **INTERNET**, haga clic en el icono de libro situado en la esquina superior derecha.

![setup wizard 1](https://static.gl-inet.com/docs/router/en/4/tutorials/internet/setup_wizard1.png){class="glboxshadow"}

Siga el asistente de configuración para configurar el router y obtener acceso a Internet.

![setup wizard 2](https://static.gl-inet.com/docs/router/en/4/tutorials/internet/setup_wizard2.png){class="glboxshadow"}

Si utiliza la [app de GL.iNet](https://www.gl-inet.com/app/#download-app-glinet){target="\_blank"}, siga las instrucciones en pantalla para configurar el acceso a Internet.
