# Filtro contenuti

Content Filter offre una protezione online intelligente basata sulla classificazione DPI, bloccando automaticamente siti web dannosi o malevoli per mantenere la rete pulita e sicura.

**Nota**: questa funzione e' attualmente disponibile solo su **GL-MT5000 (Brume 3)**.

---

Sul lato sinistro del pannello di amministrazione web, vai su **FLOW CONTROL** > **Content Filter**.

Personalizza i contenuti bloccati, come app, domini o indirizzi IP, quindi fai clic su **Apply**.

![content filter](https://static.gl-inet.com/docs/router/en/4/interface_guide/content_filter/content-filter.png){class="glboxshadow"}

- **Blocked Apps List**: questa sezione include tre categorie preselezionate: Gambling, Adult e Malware. Quando sono abilitate, tutti i siti web, servizi o app correlati a queste tre categorie verranno bloccati. Se necessario, puoi fare clic su Edit App per aggiungere altre categorie, ad esempio Game, Social Media e Shopping.

- **Blocked Domain / IP List**: questa sezione consente di inserire manualmente domini specifici (ad esempio `google.com`) o indirizzi IP (ad esempio `192.168.8.0/24`) per bloccarne l'accesso. L'elenco supporta fino a 10000 voci.
