# GL.iNetルーターのファームウェアをダウンロードする方法

ルーターをアップデートしたい場合（またはU-Boot経由でブリック問題を解決したい場合）、以下の手順でファームウェアファイルをダウンロードします：

1. [GL.iNetファームウェアダウンロードセンター](https://dl.gl-inet.com)で、検索バーからルーターモデルを選択します。
2. 上部のタブを選択します：
    * **Stable（安定版）**：日常使用の公式リリースで、ほとんどのユーザーに推奨されます。
    * **Beta（ベータ版）**（特定のルーターモデルのみ利用可能）：まだ開発中のプレリリースバージョンです。
    * **Snapshot（スナップショット）**（特定のルーターモデルのみ利用可能）：最新のアップデートを含むデイリービルドです。
3. **Download（ダウンロード）**ボタンをクリックします。
