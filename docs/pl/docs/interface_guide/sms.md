# SMS

Routery komórkowe GL.iNet są wyposażone we wbudowany modem obsługujący wysyłanie i odbieranie wiadomości SMS po włożeniu karty SIM.

**Uwaga**: Ta funkcja działa wyłącznie na modelach GL.iNet 4G/5G z oryginalnym modułem 4G LTE/5G NR. Nie jest obsługiwana na innych modułach ani żadnych modułach USB.

## Szybka konfiguracja

Poniższy przykład dotyczy modelu GL-X3000 (Spitz AX).

Najpierw skonfiguruj kartę SIM, postępując zgodnie z [tą instrukcją](internet_cellular.md).

W lewym panelu bocznym panelu administracyjnego przejdź do sekcji **INTERNET** -> **Cellular**.

Kliknij ikonę koperty w prawym górnym rogu.

![sms](https://static.gl-inet.com/docs/router/en/4/interface_guide/internet_cellular/4.8/sms.png){class="glboxshadow gl-90-desktop"}

Zostaniesz przekierowany na stronę SMS, gdzie możesz przeglądać odebrane wiadomości lub wysyłać nowe.

![sms receive](https://static.gl-inet.com/docs/router/en/4/tutorials/sms/sms_receive.jpg){class="glboxshadow gl-90-desktop"}

Kliknij wiadomość, aby wyświetlić jej szczegóły.

![sms detail](https://static.gl-inet.com/docs/router/en/4/tutorials/sms/sms_detail.png){class="glboxshadow"}

Kliknij przycisk **Send Message** w prawym górnym rogu, aby napisać nową wiadomość.

![sms new message](https://static.gl-inet.com/docs/router/en/4/tutorials/sms/sms_new_message.jpg){class="glboxshadow"}

* Odbiorca

    Wybierz międzynarodowy numer kierunkowy i wpisz numer telefonu.

* From modem (SIM)

    Wybierz, z której karty SIM chcesz wysłać wiadomość.

* SMS Content

    Każda wiadomość SMS może zawierać maksymalnie 160 znaków.
