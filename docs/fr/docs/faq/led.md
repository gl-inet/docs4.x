## Routeur cellulaire

### GL-XE3000

![gl-xe3000 led](https://static.gl-inet.com/docs/router/en/4/user_guide/gl-xe3000/hardware_info/xe3000led.jpg){class="glboxshadow gl-70-desktop"}

* LED d'alimentation
	* Vert : sous tension
	* Éteinte : hors tension

* LED de batterie (pendant la charge)
	* Rouge clignotant : niveau de batterie 0 %-20 %
	* Violet clignotant : niveau de batterie 21 %-50 %
	* Vert clignotant : niveau de batterie 51 %-99 %
	* Vert fixe : niveau de batterie 100 %

* LED de batterie (hors charge)

	Lorsqu'il n'est pas en charge, le voyant de batterie s'allume pendant 5 secondes après avoir appuyé sur le bouton d'alimentation.

	* Rouge : niveau de batterie 0 %-20 %
	* Violet : niveau de batterie 21 %-50 %
	* Vert : niveau de batterie 51 %-100 %
	
* LED Internet
	* Allumée : Internet est connecté
	* Éteinte : Internet est déconnecté

* LED Wi-Fi 2.4G
	* Allumée : le Wi-Fi 2.4G est prêt
	* Clignotante : le Wi-Fi 2.4G communique

* LED Wi-Fi 5G
	* Allumée : le Wi-Fi 5G est prêt
	* Clignotante : le Wi-Fi 5G communique

* LED cellulaire
	* Connexion au réseau mobile établie

## Passerelles de sécurité et mini-routeurs

### GL-MT5000

![gl-mt5000 interface](https://static.gl-inet.com/docs/router/en/4/user_guide/gl-mt5000/hardware_info/mt5000_interface.png){class="glboxshadow"}

* LED système
	* Bleu : démarrage du système
	* Bleu clignotant : recherche du réseau
	* Blanc : fonctionnement normal
* LED VPN
	* Éteinte : le client VPN n'est pas connecté
	* Clignotante : le client VPN est en cours de connexion
	* Fixe : le client VPN est connecté
	
Remarque : l'état du serveur VPN n'est pas affiché sur la LED.

### GL-MT300N-V2

![gl-mt300n-v2 interface](https://static.gl-inet.com/docs/router/en/4/user_guide/gl-mt300n-v2/product_info/mt300n-v2_interface.png){class="glboxshadow"}

- LED d'alimentation
	- Allumée : sous tension
	- Éteinte : hors tension

- LED centrale (configurable)
	- Allumée : cette LED a été personnalisée
	- Éteinte : cette LED n'a pas été personnalisée (mode par défaut)

- LED sans fil
	- Allumée : le Wi-Fi 2.4G est connecté
	- Éteinte : le Wi-Fi 2.4G est déconnecté

	Pour les premiers lots, la LED sans fil peut s'afficher en rouge au lieu de blanc en fonctionnement normal.
