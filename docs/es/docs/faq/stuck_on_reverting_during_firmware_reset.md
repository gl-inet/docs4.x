# Bloqueado en "Reverting" durante el restablecimiento del firmware

Si su router se queda bloqueado en "Reverting" durante un restablecimiento de firmware, siga los pasos que se indican a continuación para resolver el problema.

![stuck on reverting](https://static.gl-inet.com/docs/router/en/4/faq/troubleshooting/stuck_on_reverting/reverting.png){class="glboxshadow"}

Después de cada paso, compruebe si el problema se ha resuelto. Si se resuelve, omita los pasos restantes.

1. Espere al menos 10 minutos para ver si el proceso de reversión se completa por sí solo. En ocasiones, este proceso puede tardar más de lo esperado.

2. Si el proceso de reversión no se completa después de 10 minutos, apague el router desconectando el cable de alimentación. Espere 30 segundos, vuelva a conectarlo y espere a que el router arranque.

3. Abra un navegador web, acceda a `192.168.8.1` e inicie sesión. Si la operación de reversión se completó correctamente, se le dirigirá a la guía de configuración.

4. Active el modo privado o incógnito de su navegador y acceda a `192.168.8.1`. Si el proceso de reversión se completó correctamente, se le dirigirá a la guía de configuración.

5. Use un navegador diferente, por ejemplo, Chrome, Edge o Safari, para acceder a `192.168.8.1`. Si el proceso de reversión se completó correctamente, se le dirigirá a la guía de configuración.

6. Si aún no puede acceder al panel de administración web del router o el router no funciona correctamente, intente restablecerlo a los valores de fábrica siguiendo [esta](repair_network_or_reset_firmware.md/#reset-to-factory) guía.
