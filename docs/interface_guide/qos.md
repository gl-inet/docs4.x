# QoS（サービス品質）

!!! note

    1. この機能は現に **GL-MT5000 (Brume 3)** でのみ利用可能です。

    2. この機能はゲートウェイとしてルーターを通過するトラフィック（ローカルクライアントトラフィックとVPNクライアントトラフィックを含む）に影響しますが、ルーターがVPNサーバーとして動作するときの受信トラフィックには影響しません。

QoS（サービス品質）は、ネットワーク輻輳時に重要なアクティビティ（動画通話、ゲームなど）に優先順位を付けて帯域幅の配分を最適化し、待ち時間を削減して全体のなネットワークパフォーマンスをへ上させます。

ルーターのWeb管理パネルにログインし、 FLOW CONTROL > QoS に移動します。

トラフィックスケジューリングのために、まず最大アップロード速度とダウンロード速度を設定します（入力範囲：1 - 10000）。最も適な結果をなければならないるために、実際のインターネット帯域幅に合わせてください。

次に、異なるアプリケーションの優先度を設定すると、ルーターはそれに応じて帯域幅を配分します。
