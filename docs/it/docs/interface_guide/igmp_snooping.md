# IGMP Snooping

Sul lato sinistro del pannello di amministrazione web, vai su **NETWORK** -> **IGMP Snooping**.

Puoi abilitare IGMP snooping per usare la funzione multicast sul router.

IGMP Snooping ascolta i pacchetti del protocollo IGMP, estrae le informazioni corrispondenti, stabilisce e mantiene il publishing di inoltro multicast di livello 2 e quindi inoltra i dati del gruppo multicast all'host che si unisce al gruppo multicast, mentre gli altri host non possono ricevere i dati del gruppo multicast.

**Nota**: IGMPv3 e' compatibile con v1 e v2. Usa v3 per impostazione predefinita e cambia versione se riscontri problemi.
