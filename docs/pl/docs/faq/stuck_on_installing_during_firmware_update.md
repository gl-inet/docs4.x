# Zatrzymanie na „Installing” podczas aktualizacji firmware

Jeśli router zatrzyma się na komunikacie „Installing” podczas aktualizacji firmware, wykonaj poniższe kroki, aby rozwiązać problem.

![stuck on installing](https://static.gl-inet.com/docs/router/en/4/faq/troubleshooting/stuck_on_installing/installing.png){class="glboxshadow"}

Po każdym kroku sprawdź, czy problem został rozwiązany. Jeśli tak, pomiń pozostałe kroki.

1. Sprawdź, czy aktualizacja firmware zakończyła się powodzeniem.

    Najpierw otwórz przeglądarkę internetową, przejdź pod adres `192.168.8.1` i zaloguj się. 
    
    Następnie sprawdź wersję firmware w lewym górnym rogu. Jeśli odpowiada wersji docelowej, aktualizacja zakończyła się sukcesem. Jeśli nie, przejdź do następnego kroku.

2. Upewnij się, że połączenie sieciowe jest stabilne, a następnie wgraj firmware ponownie.

3. Użyj innej przeglądarki (np. Edge, Safari), aby ponownie wgrać firmware.
