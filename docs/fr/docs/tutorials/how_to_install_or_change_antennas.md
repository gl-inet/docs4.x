# Comment installer ou remplacer des antennes externes pour les routeurs cellulaires ?

Ce tutoriel explique comment installer et remplacer les antennes externes sur certains routeurs cellulaires.

## Installer des antennes externes

### Mudi 7

Mudi 7 (GL-E5800) est équipé de deux ports d’antenne TS‑9 compatibles avec les fréquences cellulaires de 600 MHz à 6 GHz. Vous pouvez y connecter des antennes externes (non incluses) pour améliorer le signal cellulaire dans les zones où la couverture est faible. Cette solution convient parfaitement aux camping-cars, sous-sols et environnements intérieurs afin d’étendre la couverture et d’assurer une connexion stable.

![e5800 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/e5800-antenna.png){class="glboxshadow"}

Les ports TS‑9 du Mudi 7 sont des connecteurs femelles (orifice interne). Ils nécessitent des antennes avec un connecteur mâle TS‑9 (broche interne).

![ts-9 connector](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/ts9_connector.png){class="glboxshadow" width="328"}

### Spitz AX / Puli AX

Spitz AX (GL-X3000) et Puli AX (GL-XE3000) sont livrés avec les antennes externes suivantes :

* 2 antennes Wi-Fi (RP-SMA, 2.4G~2.5GHz, 5.15~5.84GHz)
* 4 antennes 5G NR (SMA, 600MHz~6GHz)

Suivez les étapes ci-dessous pour installer les antennes externes.

1. Fixez les antennes (dans n’importe quel ordre).

    * Fixez les deux antennes Wi-Fi 6 identiques sur les ports **Wi-Fi 1** et **Wi-Fi 2**.
    * Fixez les quatre antennes 5G NR identiques sur **MAIN**, **DIV**, **MIMO 1** et **MIMO2**.

    ![x3000 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x3000-antennas.jpg){class="gl-80-desktop glboxshadow"}

2. Orientez les antennes vers le haut pour obtenir les meilleures performances.

### Spitz Plus

Spitz Plus (GL-X2000) est livré avec les antennes externes suivantes :

* 2 antennes Wi-Fi (RP-SMA, 2.4G~2.5GHz, 5.15~5.84GHz)
* 2 antennes 4G (SMA, 600MHz~3.8GHz)

Suivez les étapes ci-dessous pour installer les antennes externes.

1. Fixez les antennes (dans n’importe quel ordre).

    - Fixez les deux antennes Wi-Fi identiques sur les ports **Wi-Fi 6**.
    - Fixez les deux antennes 4G identiques sur le port **4G**.

    ![x2000 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x2000-back.png){class="gl-80-desktop glboxshadow"}

2. Orientez les antennes vers le haut pour obtenir les meilleures performances.

### Spitz

Spitz (GL-X750/GL-X750V2) est livré avec 2 antennes 4G externes (SMA, 700M~2700MHz).

Suivez les étapes ci-dessous pour installer les antennes externes.

1. Fixez les deux antennes 4G identiques sur les ports d’antenne.

    ![x750v2 antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x750v2_back.png){class="gl-80-desktop glboxshadow"}

2. Orientez les antennes vers le haut pour obtenir les meilleures performances.

### Collie

Collie (GL-X300B-RS485) est livré avec les antennes suivantes :

* 1 antenne 4G externe
* 2 antennes Wi-Fi externes

Suivez les étapes ci-dessous pour installer les antennes.

1. Fixez les antennes (dans n’importe quel ordre).

    - Fixez les deux antennes Wi-Fi identiques sur les ports **2.4G**.
    - Fixez l’antenne 4G unique sur le port **4G**.

    ![x300b antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x300b-rs485.png){class="gl-80-desktop glboxshadow"}

    ![x300b antennas](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/x300b-rs485-back.png){class="gl-80-desktop glboxshadow"}

2. Orientez les antennes vers le haut pour obtenir les meilleures performances.

## Remplacer des antennes externes

Les instructions ci-dessous s’appliquent à tous les routeurs cellulaires dotés de ports d’antenne externe, à l’exception du Mudi 7 (GL-E5800). Cliquez [ici](#mudi-7){target="_blank"} pour consulter l’installation des antennes externes sur le Mudi 7.

Vous pouvez acheter des antennes Wi-Fi ou 4G/5G NR supplémentaires pour remplacer celles fournies dans l’emballage et ainsi adapter votre installation réseau à vos besoins.

Cependant, nous recommandons d’utiliser les antennes d’origine, car elles ont été largement testées avant expédition afin de garantir des performances et une compatibilité optimales.

Si vous préférez utiliser des antennes tierces, reportez-vous aux sections ci-dessous afin de choisir des antennes avec le bon type de connecteur.

**Conseil** : si vous avez acheté des antennes avec le mauvais connecteur, vous pouvez envisager d’utiliser un adaptateur permettant de convertir les connecteurs (par ex. SMA-RP mâle vers SMA femelle).

### Antennes Wi-Fi

La base de l’antenne Wi-Fi utilise un **connecteur femelle RP-SMA** (filetage externe et broche interne). Elle nécessite une antenne Wi-Fi avec un **connecteur mâle RP-SMA** (filetage interne et orifice interne).

![rp-sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/rp-sma-connectors.jpg){class="gl-80-desktop glboxshadow"}

### Antennes 5G NR

La base de l’antenne 5G NR utilise un **connecteur femelle SMA** (filetage externe et orifice interne). Elle nécessite une antenne 5G NR avec un **connecteur mâle SMA** (filetage interne et broche interne).

![sma connectors](https://static.gl-inet.com/docs/router/en/4/tutorials/change_x3000_xe3000_antennas/sma-connectors.jpg){class="gl-80-desktop glboxshadow"}
